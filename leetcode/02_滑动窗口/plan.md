# 02 滑动窗口

## 核心思路

滑动窗口本质是双指针的进阶版，维护一个满足条件的窗口，右指针扩张、左指针收缩。

**万能模板**：
```
left = 0
for right in range(len(s)):
    窗口加入 s[right]
    while 窗口不满足条件:
        窗口移除 s[left]
        left += 1
    更新答案
```

两种类型：
1. **定长窗口**：窗口大小固定，右指针每走一步左指针跟一步
2. **变长窗口**：窗口大小可变，求最小/最大满足条件的窗口
