# 24 组织通用管理

## 一、一句话理解
组织通用管理就是"企业日常运营"——人力资源、流程、知识、市场营销，知道企业除了项目之外还怎么运转。

## 二、核心概念与框架

| 核心概念 | 说明 |
|---------|------|
| 人力资源管理 | 招聘、培训、绩效、薪酬 |
| 流程管理 | 业务流程的设计、执行、优化 |
| 知识管理 | 知识的获取、存储、共享、应用 |
| 市场营销 | 产品推广、客户关系 |

## 三、关键知识点逐个讲

### 知识点1：人力资源管理
- **什么意思**：企业对人的管理。
- **为什么重要**：人才是企业最重要的资源。
- **考试怎么考**：考招聘、培训、绩效、薪酬等基本概念。

### 知识点2：流程管理
- **什么意思**：对企业业务流程的管理。
- **为什么重要**：流程决定效率。
- **考试怎么考**：考BPM（业务流程管理）、流程优化方法。

### 知识点3：知识管理
- **什么意思**：对企业知识资产的管理。
- **为什么重要**：知识是企业的无形资产。
- **考试怎么考**：考知识管理过程、知识管理工具。

### 知识点4：市场营销
- **什么意思**：产品推广和客户关系管理。
- **为什么重要**：市场决定企业生存。
- **考试怎么考**：考4P（产品、价格、渠道、促销）、CRM。

## 四、与其他模块的联动

### 与"资源管理"的联动
**场景**：组织人力资源管理是项目资源管理的基础。
**处理**：项目资源管理依赖组织的人力资源体系。

## 五、常见考试出题点

- **选择题一般怎么挖坑**：
  - 4P营销组合
  - 知识管理过程

- **案例分析一般怎么考**：
  - 较少考
