# 05 哈希表

## 核心思路

哈希表的核心价值是 **O(1) 查找**，把"找某个元素"从 O(n) 降到 O(1)。

常见套路：
1. **两数之和模式**：遍历时查 complement 是否在哈希表中
2. **计数统计**：用哈希表统计频率，再按频率处理
3. **前缀和+哈希**：把前缀和存入哈希，O(1) 判断子数组和
