# 论说文

> 学习日期：2026-05-26 生成
> 预计用时：4-6 小时（分 2-3 次学完）
> 分值占比：35 分（管理联考写作部分）
> 要求：700 字左右，35 分钟完成
> 核心：观点明确 + 论证充分 + 结构清晰

---

## 一、题型概述

给你一段材料（寓言、名言、社会现象等），要求你：
1. 提炼观点
2. 围绕观点展开论证
3. 联系实际，提出建议

**评分标准**：
- 立意分（10分）：观点是否正确、深刻
- 论证分（12分）：论据是否充分、论证是否严密
- 结构分（8分）：层次是否清晰
- 语言分（5分）：表达是否流畅

**时间分配**：
| 阶段 | 时间 | 内容 |
|------|------|------|
| 审题立意 | 5分钟 | 读材料、提炼观点 |
| 列提纲 | 3分钟 | 确定结构和论据 |
| 写作 | 25分钟 | 按提纲写作 |
| 检查 | 2分钟 | 检查错别字、语句通顺 |

## 二、审题立意（最关键一步）

### 审题方法

1. **找关键词**：材料中反复出现的词
2. **找矛盾点**：材料中的对立/冲突
3. **找转折点**："但是""然而"后面的内容往往是重点
4. **找结果**：故事的结局往往暗示立意方向

### 立意原则

| 原则 | 说明 | 反例 |
|------|------|------|
| 正面积极 | 观点要积极向上，符合主流价值观 | "社会不公，努力无用" |
| 小切口 | 不要面面俱到，抓住一个角度深入 | "论人生"（太宽） |
| 有深度 | 不能停留在表面，要揭示本质 | "乌鸦很傻"（太浅） |
| 可论证 | 观点要有足够的论据支撑 | "命运天注定"（无法论证） |

### 常见立意方向

| 材料类型 | 立意方向 | 禁忌方向 |
|----------|----------|----------|
| 个人成长 | 坚持、自律、学习、勇气 | 躺平、放弃 |
| 人际关系 | 信任、合作、包容、沟通 | 封闭、猜忌 |
| 企业管理 | 创新、责任、诚信、远见 | 短视、欺骗 |
| 社会现象 | 公平、规则、变革、传承 | 极端、偏激 |

### 审题立意训练

**材料 1**：
> 一只蝴蝶挣扎着从茧里出来，一个人看到后很心疼，用剪刀帮蝴蝶剪开了茧。蝴蝶出来了，但翅膀软弱无力，再也飞不起来了。

立意分析：
- 关键词：挣扎、帮助、飞不起来
- 矛盾点：好心帮忙反而害了蝴蝶
- 立意：**成长需要经历磨砺，过度保护反害之** / **逆境是成长的必经之路**

**材料 2**：
> 两个推销员去非洲卖鞋。一个看到当地人不穿鞋，沮丧地说"没有市场"；另一个兴奋地说"太好了，人人都需要鞋！"

立意分析：
- 关键词：卖鞋、不穿鞋、沮丧/兴奋
- 矛盾点：同一事实，不同态度
- 立意：**心态决定视角，视角决定成败** / **换个角度看问题**

**材料 3**：
> 有人问三个砌砖工人："你在做什么？"第一个说"砌砖"，第二个说"赚钱养家"，第三个说"建造世界上最美的教堂"。多年后，第三个成了著名建筑师。

立意分析：
- 关键词：砌砖、不同回答、建筑师
- 矛盾点：同样的工作，不同的格局
- 立意：**格局决定结局** / **远大目标引领卓越**

**材料 4**：
> 一根筷子容易折断，一把筷子折不断。

立意分析：
- 关键词：一根/一把、折断/折不断
- 矛盾点：个体力量弱 vs 集体力量强
- 立意：**团结就是力量** / **合作共赢**

**材料 5**：
> 青蛙被放进沸水会跳出来，放进冷水慢慢加热却会被煮死。

立意分析：
- 关键词：沸水/冷水、跳出来/煮死
- 矛盾点：剧烈变化有警觉，缓慢变化无察觉
- 立意：**居安思危，防微杜渐** / **警惕温水煮青蛙效应**

## 三、写作结构

### 结构一：递进式（推荐，最容易写好）

```
第一段：引出观点（100字）
第二段：为什么需要___（150字）
第三段：如何做到___（150字）
第四段：不___会怎样（150字）
第五段：总结升华（100字）
```

**优点**：逻辑层层深入，阅卷老师一目了然

### 结构二：并列式

```
第一段：引出观点（100字）
第二段：分论点一 + 论据（150字）
第三段：分论点二 + 论据（150字）
第四段：分论点三 + 论据（150字）
第五段：总结升华（100字）
```

**优点**：论据丰富，说服力强

### 结构三：正反对比式

```
第一段：引出观点（100字）
第二段：正面论证（200字）
第三段：反面论证（200字）
第四段：总结升华（100字）
```

**优点**：对比鲜明，论证有力

### 结构选择建议

| 你的情况 | 推荐结构 |
|----------|----------|
| 论据储备丰富 | 并列式 |
| 逻辑思维强 | 递进式 |
| 正反事例都有 | 正反对比式 |
| 不确定 | 递进式（最稳妥） |

## 四、开头写法（5种）

### 1. 引用式
> 古人云："___。"这句话告诉我们___。

**例**：
> 古人云："锲而不舍，金石可镂。"这句话告诉我们，坚持是通往成功的必经之路。

### 2. 设问式
> ___真的___吗？在我看来，___。

**例**：
> 失败真的只是失败吗？在我看来，失败是成功之母，是通往成功的阶梯。

### 3. 对比式
> 有人认为___，也有人认为___。我认为___。

**例**：
> 有人认为竞争才能进步，也有人认为合作才能共赢。我认为，竞争与合作并非对立，而是相辅相成。

### 4. 故事式
> ___的故事告诉我们___。

**例**：
> 勾践卧薪尝胆的故事告诉我们，逆境中的坚持终将迎来转机。

### 5. 直入式
> 面对___，我认为___。

**例**：
> 面对快速变化的时代，我认为唯有不断学习，才能立于不败之地。

## 五、论据使用

### 论据类型

| 类型 | 说明 | 效果 | 使用频率 |
|------|------|------|----------|
| 名言警句 | 权威人士的话 | 增强说服力 | 每篇1-2处 |
| 历史典故 | 历史人物/事件 | 有文化底蕴 | 每篇1-2处 |
| 现实事例 | 当代人物/企业 | 贴近现实 | 每篇1-2处 |
| 数据事实 | 统计数据 | 客观有力 | 可选 |

### 论据使用原则

1. **一个论据一段**，不要堆砌
2. **叙议结合**：先叙述论据，再分析论证（叙:议 = 1:2）
3. **论据与观点紧密相关**，不能牵强
4. **古今中外搭配**，不要只用一种类型
5. **论据要简洁**，叙述不超过3句话，重点在分析

### 常用论据模板

**正面论据**：
```
___（人物/企业）在___（情境）中，___（做了什么），最终___（结果）。
这正说明，___（回到论点）。
```

**反面论据**：
```
反观___（人物/企业），因为___（没做到什么），导致___（不良结果）。
这警示我们，___（回到论点）。
```

### 叙议结合示范

❌ **只叙不议**：
> 华为坚持自主研发芯片，经过多年努力，终于突破了技术封锁。

✅ **叙议结合**：
> 华为坚持自主研发芯片，经过多年努力，终于突破了技术封锁。正是因为华为在顺境中保持清醒、在逆境中坚守初心，才有了今天的成就。这告诉我们，唯有坚持自主创新，才能在风云变幻中立于不败之地。

## 六、结尾写法（4种）

### 1. 呼吁式
> 让我们___，在___的道路上不断前行。

**例**：
> 让我们怀揣远大的格局，在追梦的道路上不断前行。

### 2. 展望式
> 相信在___的指引下，我们终将___。

**例**：
> 相信在坚持的信念指引下，我们终将抵达成功的彼岸。

### 3. 引用式
> 正如___所说："___。"愿我们都能___。

**例**：
> 正如荀子所说："锲而不舍，金石可镂。"愿我们都能在坚持中成就自我。

### 4. 升华式
> ___不仅是一种___，更是一种___。它将引领我们走向___。

**例**：
> 创新不仅是一种能力，更是一种勇气。它将引领我们走向更加美好的未来。

## 七、论证技巧进阶

### 1. 假设论证

用"如果...就..."来加强论证力度：

> 如果没有坚持，爱迪生怎能在经历上千次失败后发明电灯？如果没有坚持，中国航天人怎能将五星红旗插上月背？

### 2. 排比论证

用排比句增强气势：

> 坚持是黑暗中的灯塔，指引我们前行；坚持是寒冬中的暖阳，给予我们力量；坚持是沙漠中的绿洲，让我们看到希望。

### 3. 对比论证

正反对比，增强说服力：

> 同样面对困境，勾践选择卧薪尝胆，最终复国；项羽选择乌江自刎，留下千古遗憾。一念之差，天壤之别。

### 4. 类比论证

用类比让抽象道理更直观：

> 人的成长如同竹子的拔节，前四年仅仅长了3厘米，但从第五年开始，每天以30厘米的速度疯长。那些看似停滞的日子，其实是在扎根。

### 5. 让步论证

先承认对方观点的合理性，再转折：

> 诚然，放弃有时可以减少损失、及时止损。然而，如果一遇困难就放弃，又怎能品尝到成功的喜悦？关键在于，坚持该坚持的，放弃该放弃的。

## 八、范文精讲

### 范文 1：寓言类——认清自我

**题目**：
> 一只老鹰从高岩上飞下来，抓起一只小羊。一只乌鸦看见了，非常羡慕，也想学老鹰抓羊。于是它猛扑到一只大公羊身上，爪子被羊毛缠住，怎么也飞不起来。牧羊人走过来，把乌鸦抓住，剪去了它的翅膀上的羽毛，送给孩子们玩耍。

**审题**：
- 关键词：羡慕、模仿、失败
- 矛盾点：乌鸦想学老鹰，但自身能力不足
- 立意：**认清自我，量力而行**

**范文**：

> **《认清自我，量力而行》**
>
> 乌鸦羡慕老鹰的威猛，盲目模仿其捕猎方式，最终落得被人捕获的下场。这则寓言启示我们：认清自我、量力而行，方能行稳致远。
>
> 认清自我是成功的前提。每个人都有自己的长处和短处，只有准确认识自身条件，才能找到适合自己的发展道路。老鹰有强健的体魄和锋利的爪子，适合捕猎；乌鸦体型小、力量弱，更适合觅食和适应环境。乌鸦若能认清自身特点，发挥灵活机敏的优势，同样可以活得精彩。然而，它被羡慕蒙蔽了双眼，忽视了自己与老鹰的本质差异，最终付出了惨痛代价。
>
> 盲目模仿则往往导致失败。现实生活中，不少企业看到行业龙头成功便急于效仿，却忽视了自身资源、能力的差异。一些创业公司照搬大公司的管理模式，结果因组织架构臃肿而丧失灵活性；一些传统企业盲目追逐互联网风口，却因缺乏技术积累而血本无归。这些案例无不说明，脱离自身实际的模仿是危险的。
>
> 当然，认清自我并非固步自封。我们应当在学习他人的基础上，结合自身特点进行创新。华为在发展初期学习西方管理经验，但并非照搬，而是结合中国文化和企业实际，形成了独特的管理体系，最终成就了今天的辉煌。这正是"学而不仿"的典范。
>
> 认清自我是一种智慧，量力而行是一种理性。唯有知己知彼，扬长避短，方能在人生和事业的道路上走得更稳、更远。

### 范文 2：名言类——坚持与放弃

**题目**：
> 有人说："坚持就是胜利。"也有人说："放弃也是一种智慧。"你怎么看？

**审题**：
- 关键词：坚持、放弃、胜利、智慧
- 矛盾点：坚持 vs 放弃，看似对立
- 立意：**坚持该坚持的，放弃该放弃的——关键在于判断**

**范文**：

> **《坚持与放弃的智慧》**
>
> "坚持就是胜利"与"放弃也是一种智慧"，两种观点看似矛盾，实则各有道理。在我看来，关键不在于坚持还是放弃，而在于能否做出正确的判断——坚持该坚持的，放弃该放弃的。
>
> 坚持是成功的基石。任何伟大的成就都离不开持之以恒的努力。袁隆平一生致力于杂交水稻研究，即使面对无数失败和外界的质疑，也从未放弃，最终解决了亿万人的吃饭问题。屠呦呦在提取青蒿素的过程中经历了190次失败，第191次才成功，正是这份坚持让她获得了诺贝尔奖。这些事例告诉我们，当方向正确、目标值得时，坚持就是通往成功的桥梁。
>
> 然而，盲目坚持同样危险。当方向错误、条件不具备时，及时放弃反而是一种智慧。诺基亚在智能手机时代来临时，固执地坚守塞班系统，不愿放弃曾经的辉煌，最终被市场淘汰。反观微软，在意识到手机操作系统无法与安卓和iOS竞争后，果断放弃这一领域，将资源转向云计算和AI，反而开辟了新的增长空间。可见，放弃不是失败，而是为了更好地出发。
>
> 那么，如何判断该坚持还是该放弃？关键在于审视目标和方向。如果目标是值得追求的、方向是正确的，那么即使过程艰难也应坚持；如果发现方向有误、继续下去只会造成更大损失，那么及时止损就是明智之举。正如古人所言："识时务者为俊杰。"
>
> 坚持与放弃，并非非此即彼的选择，而是需要智慧的判断。愿我们都能在人生的关键时刻，做出正确的抉择——坚持该坚持的，放弃该放弃的，走出属于自己的精彩人生。

### 范文 3：社会现象类——创新

**题目**：
> 近年来，许多传统行业面临转型压力。有人认为应该坚守传统，因为传统是根基；也有人认为必须创新求变，因为时代在发展。对此，你怎么看？

**审题**：
- 关键词：传统、创新、根基、发展
- 矛盾点：守传统 vs 求创新
- 立意：**传承不守旧，创新不忘本——在传承中创新**

**范文**：

> **《在传承中创新，在创新中传承》**
>
> 面对传统与创新的争论，我认为两者并非对立，而是相辅相成。传承是创新的根基，创新是传承的延续。唯有在传承中创新，方能行稳致远。
>
> 传承是创新的根基。没有传承，创新便成了无源之水。故宫博物院在开发文创产品时，始终以深厚的文化底蕴为基础，将六百年的宫廷文化融入现代设计，让"故宫口红""故宫日历"成为现象级产品。正是因为对传统文化的深刻理解和尊重，故宫的创新才有了灵魂和底气。反观一些企业，为了创新而创新，抛弃了自身最核心的优势，结果既丢了传统又没换来新意，得不偿失。
>
> 创新是传承的延续。没有创新，传承便成了刻舟求剑。柯达发明了数码相机技术，却因为固守胶卷业务的利润而错失转型良机，最终走向破产。同样，百年来以手工著称的瑞士钟表业，在石英危机中几近消亡，正是通过将传统工艺与智能技术相结合，才重新焕发了生机。这些案例告诉我们，传统若不与时俱进，终将被时代淘汰。
>
> 在传承中创新，需要智慧与勇气。华为在5G领域的突破，既传承了数十年通信技术的积累，又在标准制定上大胆创新，最终引领了行业发展。比亚迪从传统电池制造起步，在核心技术积累的基础上向新能源汽车转型，将"技术为王"的传统与"绿色出行"的创新完美结合。
>
> 传承不是守旧，创新不是颠覆。正如古语所言："苟日新，日日新，又日新。"让我们在传承中汲取力量，在创新中开辟未来，让传统在创新中焕发新的生命力。

### 范文 4：寓言类——格局与目标

**题目**：
> 有人问三个砌砖工人："你在做什么？"第一个说"砌砖"，第二个说"赚钱养家"，第三个说"建造世界上最美的教堂"。多年后，第三个成了著名建筑师。

**审题**：
- 关键词：砌砖、不同回答、建筑师
- 矛盾点：同样的工作，不同的格局
- 立意：**格局决定结局，目标引领卓越**

**范文**：

> **《格局决定结局》**
>
> 三个砌砖工人，同样的工作，不同的回答，最终走向了不同的人生。这则故事启示我们：格局决定结局，目标的高度决定了人生的高度。
>
> 格局的大小决定了视野的宽窄。第一个工人只看到了眼前的砖块，他的世界被局限在一砖一瓦之中；第二个工人看到了家庭的责任，视野稍宽但仍然有限；第三个工人看到了宏伟的建筑，他的目光穿越了眼前的辛苦，投向了远方的蓝图。正是这种不同的视野，决定了他们日后不同的成就。正如王之涣所言："欲穷千里目，更上一层楼。"站得越高，看得越远，走得也越远。
>
> 目标的高度决定了努力的方向。当一个人只把工作当作谋生手段时，他满足于完成任务；当一个人将工作与远大目标相连时，他会不断追求卓越。任正非在华为创立之初就立志"做世界一流的通信企业"，正是这个远大目标引领华为从一个小作坊成长为全球科技巨头。如果任正非的目标只是"赚点小钱"，华为不可能有今天的成就。
>
> 当然，大格局不是好高骛远，而是在脚踏实地的同时仰望星空。第三个工人并非空谈理想，他同样在砌砖，只是他的每一块砖都有了意义。正如老子所言："千里之行，始于足下。"远大的目标需要从脚下的每一步开始，但心中有了方向，每一步就有了力量。
>
> 格局是一种眼界，更是一种力量。愿我们都能像第三个工人一样，在平凡的岗位上心怀远大，用大格局书写不平凡的人生。

### 范文 5：辩证类——竞争与合作

**题目**：
> 在现代社会中，竞争无处不在。有人说，竞争推动进步；也有人说，合作才能共赢。请谈谈你的看法。

**审题**：
- 关键词：竞争、合作、进步、共赢
- 矛盾点：竞争 vs 合作
- 立意：**竞争与合作相辅相成，良性竞争促进进步，合作共赢创造更大价值**

**范文**：

> **《竞争与合作：鸟之双翼》**
>
> 竞争与合作，看似水火不容，实则如鸟之双翼，缺一不可。良性竞争激发活力，真诚合作创造共赢，二者相辅相成，共同推动社会进步。
>
> 竞争是进步的催化剂。没有竞争，就没有动力；没有压力，就没有突破。中国高铁从引进技术到自主创新，正是在与国际巨头的竞争中不断超越自我，最终成为世界高铁的领跑者。奥运赛场上，运动员们在竞争中不断刷新纪录，将人类的体能极限一次次推向新的高度。竞争让人走出舒适区，激发潜能，创造奇迹。
>
> 然而，恶性竞争则两败俱伤。当竞争变成你死我活的零和博弈，结果往往是双输。共享单车行业的混战就是前车之鉴——各品牌疯狂烧钱、无序投放，最终多数企业倒闭，留下满地废铁和巨额押金欠款。这种没有底线的竞争，不仅没有推动行业进步，反而造成了巨大的资源浪费。
>
> 合作是共赢的桥梁。在许多领域，合作比竞争能创造更大的价值。开源软件运动让全球开发者共同协作，创造了Linux等伟大产品；国际空间站是多个国家合作的结晶，任何一国都无法独立完成。在商业领域，苹果与供应链企业的深度合作，让每一部iPhone都凝聚了全球智慧。
>
> 竞争与合作并非对立，而是辩证统一。竞争中可以有合作——企业之间在标准制定、行业规范上合作，在产品和服务上竞争；合作中也可以有竞争——团队成员为共同目标努力的同时，也在良性竞争中提升自我。正如古人所言："君子和而不同。"在竞争中保持合作的胸怀，在合作中保持竞争的锐气，方能行稳致远。

## 九、高频主题写作框架

### 主题 1：坚持/毅力

**递进式框架**：
```
开头：坚持是通往成功的必经之路
为什么：任何成就都离不开持之以恒（举例：袁隆平、勾践）
怎么做：明确目标、保持信念、克服困难
不坚持会怎样：半途而废、功亏一篑（举例：项羽）
结尾：坚持是一种力量，更是一种信仰
```

### 主题 2：创新/变革

**递进式框架**：
```
开头：创新是时代发展的引擎
为什么：墨守成规终将被淘汰（举例：诺基亚、柯达）
怎么做：敢于突破、善于学习、勇于实践（举例：华为、比亚迪）
不变革会怎样：固步自封、失去竞争力
结尾：创新是一种勇气，更是一种责任
```

### 主题 3：诚信/品格

**递进式框架**：
```
开头：诚信是立身之本
为什么：诚信赢得信任，信任创造价值（举例：商鞅立木）
怎么做：言行一致、坚守承诺、不为利诱
不诚信会怎样：失信于人、众叛亲离（举例：某些企业造假）
结尾：诚信是一种品格，更是一种力量
```

### 主题 4：远见/格局

**递进式框架**：
```
开头：格局决定结局
为什么：远见卓识让人抢占先机（举例：华为芯片布局）
怎么做：着眼长远、跳出局限、持续学习
没有远见会怎样：目光短浅、错失机遇（举例：柯达）
结尾：远见是一种智慧，更是一种境界
```

### 主题 5：团结/合作

**递进式框架**：
```
开头：团结就是力量
为什么：个人力量有限，团队力量无穷（举例：管鲍之交）
怎么做：信任他人、包容差异、协同配合
不团结会怎样：一盘散沙、各自为战
结尾：团结是一种精神，更是一种力量
```

## 十一、易错点分析

### 易错点 1：立意偏题

❌ **错误**：材料讲"认清自我"，写成"努力奋斗"
✅ **正确**：紧扣材料关键词立意

> **提醒**：审题时圈出关键词，立意必须包含关键词

### 易错点 2：立意太浅

❌ **错误**：材料讲乌鸦模仿老鹰，立意为"乌鸦很傻"
✅ **正确**：立意为"认清自我，量力而行"

> **提醒**：从具体事例中提炼普遍道理，不要停留在表面

### 易错点 3：只叙不议

❌ **错误**：讲了一个故事就完了，没有分析
✅ **正确**：叙述论据后，分析论据如何支撑观点

> **提醒**：叙议比例约 1:2，分析比叙述更重要

### 易错点 4：论据与观点脱节

❌ **错误**：观点是"创新"，论据讲的是"坚持"
✅ **正确**：论据必须直接支撑观点

> **提醒**：写完论据后，用一句话点明论据与观点的关系

### 易错点 5：结构混乱

❌ **错误**：想到哪写到哪，没有逻辑层次
✅ **正确**：先列提纲，按结构写作

> **提醒**：花3分钟列提纲，磨刀不误砍柴工

### 易错点 6：语言口语化

❌ **错误**："我觉得吧，这个事挺重要的"
✅ **正确**："由此可见，这一问题不容忽视"

> **提醒**：用书面语，避免口语化表达

### 易错点 7：字数超标或不足

❌ **错误**：写了 900 字或只有 500 字
✅ **正确**：控制在 650-750 字

> **提醒**：每段预设字数，开头结尾各100字，中间3段各150-200字

### 易错点 8：观点骑墙

❌ **错误**："坚持也好，放弃也好，都有道理"
✅ **正确**：明确表态，可以有辩证但要有主次

> **提醒**：观点必须明确，可以辩证但不能模棱两可

## 十二、语言提升技巧

### 1. 用四字词语提升文采

| 普通表达 | 文采表达 |
|----------|----------|
| 很难做到 | 知易行难 |
| 非常重要 | 举足轻重 |
| 慢慢积累 | 日积月累 |
| 一起努力 | 同舟共济 |
| 不怕困难 | 迎难而上 |
| 找到方法 | 对症下药 |
| 互相帮助 | 守望相助 |
| 坚持到底 | 矢志不渝 |
| 抓住机会 | 乘势而上 |
| 不断创新 | 推陈出新 |

### 2. 用对偶句增强节奏

> 传承是根，创新是魂。
> 守正为本，创新为翼。
> 脚踏实地，仰望星空。

### 3. 用比喻让道理更生动

> 坚持如同磨刀石，磨砺出锋利的刀刃。
> 创新如同破冰船，开辟出前行的航道。
> 诚信如同基石，支撑起信任的大厦。

### 4. 常用过渡句

| 位置 | 过渡句 |
|------|--------|
| 开头→论证 | 这则故事启示我们 / 这一现象引发我们深思 |
| 正面→反面 | 反观 / 然而 / 与之形成鲜明对比的是 |
| 论证→总结 | 由此可见 / 综上所述 / 毋庸置疑 |
| 让步→转折 | 诚然...然而... / 固然...但是... |
