# 综合推理

> 学习日期：2026-05-26 生成
> 预计用时：6-8 小时（分 3-4 次学完）
> 分值占比：约 10 题，20 分
> 特点：信息量大，需要画表/列条件，耗时最长
> ⚠️ 综合推理是逻辑部分最耗时的板块，掌握方法后可大幅提速

---

## 一、解题通用方法

### 1. 信息整理

**第一步永远是整理条件**，不要在脑子里转，必须写下来：

- 用表格整理人物-属性对应关系
- 用数轴整理大小/顺序关系
- 用符号整理逻辑关系（→、∧、∨）

### 2. 排除法

- 从限制最严的条件入手
- 先确定能确定的，再逐步排除

### 3. 假设法

- 当条件不足以直接推出时，假设某条件成立
- 如果推出矛盾，则假设不成立
- 如果不矛盾，继续推理

### 4. 时间分配策略

| 题型 | 建议用时 | 策略 |
|------|----------|------|
| 排序题 | 2-3 分钟 | 画数轴，标位置 |
| 分组题 | 2-3 分钟 | 画框架，标限制 |
| 匹配题 | 2-3 分钟 | 画表格，排除法 |
| 真假话 | 1-2 分钟 | 找矛盾对 |
| 连锁推理 | 1-2 分钟 | 串联条件 |
| 复杂综合 | 3-4 分钟 | 多法并用 |

> **核心原则**：单题超过 3 分钟做不出，先跳过，回头再做

## 二、排序题

### 题型特征
> A、B、C、D、E 五人排队，已知某些人的相对位置，求排列方式。

### 解题步骤
1. 列出所有位置（1-5）
2. 标注确定条件
3. 标注相对条件（A 在 B 前面）
4. 找限制最严的条件入手

### 常见条件类型

| 条件类型 | 例子 | 符号化 |
|----------|------|--------|
| 确定位置 | A 在第 3 位 | A=3 |
| 不在某位置 | A 不在两端 | A≠1, A≠5 |
| 相对顺序 | A 在 B 左边 | A...B |
| 相邻 | A 和 B 相邻 | AB 或 BA |
| 间隔 | A 和 B 之间隔 1 人 | A_B 或 B_A |
| 紧挨 | A 紧挨 B 右边 | BA |

### 例题 1

> 五人 A、B、C、D、E 从左到右排一排：
> (1) A 不在两端
> (2) B 在 C 的左边（不一定相邻）
> (3) D 紧挨着 E 的右边
> (4) A 在 D 的左边

**解析**：
由 (3)：DE 相邻，D 在 E 右边 → ...DE...
由 (4)：A 在 D 左边 → A...DE...
由 (2)：B 在 C 左边 → B...C...
由 (1)：A 不在位置 1 和 5

尝试排列：
B, A, C, D, E → 检查：(1) A 在位置2 ✓ (2) B在C左 ✓ (3) DE相邻 ✓ (4) A在D左 ✓

**答案：BACDE**（可能有多解，需结合选项）

### 例题 2

> 6 本书编号 1-6 从左到右排列：
> (1) 数学书不在两端
> (2) 物理书在化学书右边
> (3) 英语书紧挨数学书左边
> (4) 语文书在位置 3

**解析**：
由 (4)：语文=3
由 (3)：英语紧挨数学左边 → EM 相邻
由 (1)：数学≠1, 数学≠6
由 (2)：物理在化学右边 → C...P

EM 相邻，数学≠1,6：
- EM 在 1,2 → 数学=2 ✓
- EM 在 2,3 → 3 是语文，冲突
- EM 在 3,4 → 3 是语文，冲突
- EM 在 4,5 → 数学=5 ✓
- EM 在 5,6 → 数学=6，冲突

所以 EM 在 1,2 或 4,5

方案1：E=1, M=2, 语文=3
剩余 4,5,6 放 C,P,历史
C...P → C 在 P 左边
C=4, P=5 或 C=4, P=6 或 C=5, P=6

方案2：语文=3, E=4, M=5
剩余 1,2,6 放 C,P,历史
C...P → C 在 P 左边
C=1, P=2 或 C=1, P=6 或 C=2, P=6

## 三、分组题

### 题型特征
> 将若干元素分成若干组，每组有容量限制和搭配限制。

### 解题步骤
1. 画出分组框架
2. 标注每组容量
3. 标注"必须同组""不能同组"条件
4. 从限制最严的元素入手

### 常见条件类型

| 条件类型 | 例子 | 处理方式 |
|----------|------|----------|
| 必须同组 | A 和 B 必须同组 | AB 绑定 |
| 不能同组 | C 和 D 不能同组 | CD 分开 |
| 某组必须有 | A 组必须有 X | 直接填入 |
| 某组不能有 | A 组不能有 Y | 排除 |
| 容量限制 | 每组恰好 3 人 | 控制总数 |

### 例题 1

> 6 人分成 2 组，每组 3 人：
> (1) A 和 B 必须同组
> (2) C 和 D 不能同组
> (3) E 不在 A 所在的组

**解析**：
由 (1)：AB 同组
由 (3)：E 不和 AB 同组 → E 在另一组
由 (2)：C 和 D 分开

组1：A, B, ?
组2：E, ?, ?

C 和 D 各去一组：
- 组1=ABC，组2=EDF
- 组1=ABD，组2=ECF

### 例题 2

> 7 人分成 3 组，人数分别为 3、2、2：
> (1) A 和 B 必须同组
> (2) C 不能和 D 同组
> (3) E 必须在 3 人组
> (4) F 和 G 不能同组

**解析**：
由 (3)：E 在 3 人组
由 (1)：AB 同组 → AB 在 3 人组（因为 2 人组放不下 AB+1）
所以 3 人组：A, B, E
由 (2)：C 和 D 分开
由 (4)：F 和 G 分开

2 人组1：C/F, D/G（C和D分开，F和G分开）
2 人组2：剩余两人

方案：
- 组1(3人)：A, B, E
- 组2(2人)：C, F
- 组3(2人)：D, G

或：
- 组1(3人)：A, B, E
- 组2(2人)：C, G
- 组3(2人)：D, F

## 四、匹配题

### 题型特征
> 多个人和多个属性一一对应，给出若干线索，求对应关系。

### 解题方法：列表法

画一个"人×属性"的表格，用 ✓ 和 ✗ 逐步填充。

### 列表法步骤

1. 画表格（人×属性）
2. 根据条件填 ✓ 和 ✗
3. 每行每列只能有一个 ✓
4. 某行/列只剩一个空位 → 直接填 ✓
5. 反复排除直到填满

### 例题 1

> 甲、乙、丙三人分别是工程师、医生、律师（不对应）：
> (1) 甲不是工程师
> (2) 乙不是医生
> (3) 工程师比丙年龄大

**解析**：

| | 工程师 | 医生 | 律师 |
|------|--------|------|------|
| 甲 | ✗(1) | | |
| 乙 | | ✗(2) | |
| 丙 | ✗(3) | | |

由 (3)：丙不是工程师（工程师比丙大，丙不能比自己大）

所以工程师只能是乙。

| | 工程师 | 医生 | 律师 |
|------|--------|------|------|
| 甲 | ✗ | | |
| 乙 | ✓ | ✗ | ✗ |
| 丙 | ✗ | | |

乙是工程师，甲和丙分医生和律师。

**答案：乙是工程师**

### 例题 2

> 四人 W、X、Y、Z 分别来自四个城市北京、上海、广州、深圳：
> (1) W 不是北京人也不是上海人
> (2) X 不是广州人
> (3) Y 是深圳人
> (4) Z 不是北京人

**解析**：

| | 北京 | 上海 | 广州 | 深圳 |
|---|------|------|------|------|
| W | ✗(1) | ✗(1) | | |
| X | | | ✗(2) | |
| Y | | | | ✓(3) |
| Z | ✗(4) | | | |

Y=深圳，所以其他人≠深圳
W≠北京，W≠上海，W≠深圳 → W=广州

| | 北京 | 上海 | 广州 | 深圳 |
|---|------|------|------|------|
| W | ✗ | ✗ | ✓ | ✗ |
| X | | | ✗ | ✗ |
| Y | ✗ | ✗ | ✗ | ✓ |
| Z | ✗ | | ✗ | ✗ |

X≠广州，X≠深圳 → X=北京或上海
Z≠北京，Z≠广州，Z≠深圳 → Z=上海

| | 北京 | 上海 | 广州 | 深圳 |
|---|------|------|------|------|
| W | ✗ | ✗ | ✓ | ✗ |
| X | | ✗ | ✗ | ✗ |
| Y | ✗ | ✗ | ✗ | ✓ |
| Z | ✗ | ✓ | ✗ | ✗ |

X=北京

**答案：W=广州，X=北京，Y=深圳，Z=上海**

## 五、真假话问题

### 题型特征
> 几个人说话，已知几真几假，求谁说真话。

### 解题方法

**方法一：找矛盾对**
- A 说"B 说假话"，B 说"A 说真话" → 矛盾对，必一真一假

**方法二：假设法**
- 假设某人说真话，看是否与已知条件矛盾

**方法三：代入法**
- 把选项代入，验证是否符合真假条件

### 例题 1

> 甲说："乙在说谎"
> 乙说："丙在说谎"
> 丙说："甲和乙都在说谎"
> 已知只有一人说真话，谁说真话？

**解析**：
假设甲说真话 → 乙说谎 → 丙说真话（因为乙说谎，丙说"甲乙都说谎"中"乙说谎"为真，但甲没说谎，所以丙说"甲乙都说谎"为假）
等等，丙说"甲和乙都在说谎" = ¬甲 ∧ ¬乙
如果甲说真话，则 ¬甲 为假，所以丙的话为假 ✓

甲真 → 乙假 → 丙假，只有1人说真话 ✓

**答案：甲说真话**

### 例题 2

> A 说："是 B 做的"
> B 说："是 D 做的"
> C 说："不是我做的"
> D 说："B 在说谎"
> 已知只有一人做了，且只有一人说真话，谁做的？

**解析**：
B 和 D 矛盾（B 说 D 做的，D 说 B 在说谎），必一真一假

所以 A 和 C 都说假话
A 假 → 不是 B 做的
C 假 → 是 C 做的

验证：C 做的
A："是B做的" → 假 ✓
B："是D做的" → 假
C："不是我做的" → 假 ✓
D："B在说谎" → 真 ✓

只有 D 说真话 ✓

**答案：C 做的**

## 六、连锁推理型

### 题型特征
> 给出多个条件命题，要求推出结论。

### 解题方法

1. 把所有条件符号化
2. 串联成推理链
3. 用肯前必肯后、否后必否前推出结论

### 例题

> 已知：
> (1) 如果甲上场，那么乙不上场
> (2) 如果丙不上场，那么丁上场
> (3) 只有乙上场，丁才不上场
> (4) 甲上场了
> 问：丙是否上场？

**解析**：
(1) 甲 → ¬乙
(2) ¬丙 → 丁
(3) ¬丁 → 乙（"只有乙，才¬丁" = 后推前 = ¬丁 → 乙）
(4) 甲

由 (4) 甲，(1) → ¬乙
¬乙 → ¬(¬丁)（由 (3) 逆否：¬乙 → ¬¬丁 = 丁）
丁 → ¬(¬丙)（由 (2) 逆否：¬丁 → ¬丙 的逆否是 丙 → 丁，但我们需要 丁 → 丙）

等等，(2) 是 ¬丙 → 丁，逆否是 ¬丁 → 丙
我们有 丁，不能从 ¬丙 → 丁 推出 ¬丙

重新分析：
我们有 丁为真
(2) ¬丙 → 丁，丁为真 → 丙不确定（肯后不确定）

所以无法确定丙是否上场。

**答案：无法确定**

## 七、数独型/矩阵型

### 题型特征
> 矩阵填空，每行每列不重复，附加额外条件。

### 解题方法

1. 画矩阵表格
2. 填入已知信息
3. 用排除法逐步填充
4. 从限制最多的行/列入手

### 例题

> 3 人（甲乙丙）参加 3 项比赛（跑步、游泳、自行车），3 天（周一、周三、周五）各比一项：
> (1) 甲不在周一比赛
> (2) 乙参加游泳
> (3) 跑步在周三
> (4) 丙不在周五比赛

**解析**：

| | 跑步(周三) | 游泳 | 自行车 |
|---|-----------|------|--------|
| 周一 | | | |
| 周三 | ✓(3) | | |
| 周五 | | | |

由 (3)：跑步在周三
由 (2)：乙参加游泳

| | 跑步(周三) | 游泳 | 自行车 |
|---|-----------|------|--------|
| 甲 | | ✗ | |
| 乙 | ✗ | ✓ | ✗ |
| 丙 | | ✗ | |

乙=游泳，甲和丙分跑步和自行车
由 (1)：甲不在周一 → 甲的比赛不在周一
由 (4)：丙不在周五 → 丙的比赛不在周五

跑步在周三：
- 如果甲=跑步 → 甲在周三比赛 ✓（不在周一 ✓）
- 如果丙=跑步 → 丙在周三比赛 ✓（不在周五 ✓）

两种都可行，需要更多信息。

假设甲=跑步（周三），则丙=自行车
乙=游泳，游泳不在周三（周三跑步），游泳在周一或周五
甲在周三 ✓
丙=自行车，不在周五 → 丙在周一
乙=游泳 → 乙在周五

验证：(1) 甲不在周一 ✓ (2) 乙游泳 ✓ (3) 跑步周三 ✓ (4) 丙不在周五 ✓

**答案：甲-跑步(周三)，乙-游泳(周五)，丙-自行车(周一)**

## 八、易错点分析

### 易错点 1：条件没写下来

❌ **错误**：在脑子里推理，遗漏条件
✅ **正确**：所有条件必须写下来，画表/列式

> **提醒**：综合推理信息量大，脑子转不过来必须写

### 易错点 2：忽略"不一定相邻"

❌ **错误**："A 在 B 左边" = AB 相邻
✅ **正确**："A 在 B 左边"只是相对顺序，中间可以有人

> **提醒**：只有"紧挨""相邻"才是紧邻

### 易错点 3：分组题忘记容量限制

❌ **错误**：3 人组里放了 4 个人
✅ **正确**：每填入一人都要检查组内人数

> **提醒**：分组题必须时刻注意容量

### 易错点 4：匹配题遗漏排除

❌ **错误**：填了一个 ✓ 就忘了排除同行同列的其他位置
✅ **正确**：每填一个 ✓，立即排除同行同列

> **提醒**：填 ✓ 后立即画 ✗

### 易错点 5：真假话问题假设不全

❌ **错误**：只假设了一种情况就下结论
✅ **正确**：必须验证所有假设，确保唯一解

> **提醒**：假设法要穷举，不能遗漏

### 易错点 6：时间失控

❌ **错误**：一道综合推理花了 10 分钟
✅ **正确**：单题不超过 3 分钟，做不出先跳

> **提醒**：综合推理最耗时，必须控制时间

### 易错点 7：混淆"必须"和"可能"

❌ **错误**：把"可能是"当成"一定是"
✅ **正确**：题目问"一定"还是"可能"，区分对待

> **提醒**："一定"需要唯一解，"可能"只需存在一种情况

### 易错点 8：连锁推理方向错误

❌ **错误**：A→B，知道 B 为真就推出 A 为真
✅ **正确**：肯后不确定，只能否后必否前

> **提醒**：连锁推理中方向很重要

### 练习 4：分组
> 8 人分成 2 组，每组 4 人：
> (1) A 和 B 必须同组
> (2) C 和 D 不能同组
> (3) E 和 F 必须同组
> (4) G 不在 A 所在的组
> (5) H 在 C 所在的组
> 问：A 所在的组有哪些人？

<details>
<summary>点击查看答案</summary>

**答案：A, B, E, F 或 A, B, C/D, ?**

解析：
由(1)：AB同组
由(4)：G不在AB组→G在另一组
由(3)：EF同组
AB组已有2人，EF组也有2人

如果AB和EF同组：ABEF=4人组 ✓
另一组：G, C/D, D/C, H

由(5)：H和C同组

### 练习 8：真假话
> A、B、C 三人，一人是老实人（总说真话），一人是骗子（总说假话），一人是普通人（有时真有时假）：
> A 说："我是老实人"
> B 说："A 说的是真话"
> C 说："B 不是老实人"
> 问：谁是老实人？

<details>
<summary>点击查看答案</summary>

**答案：C**

解析：
假设A是老实人→A说真话→"我是老实人"为真✓
B说"A说真话"→B也说真话
但只有一人是老实人，A和B都说真话矛盾

假设B是老实人→B说真话→"A说真话"→A也是老实人，矛盾

假设C是老实人→C说真话→"B不是老实人"为真✓
A说"我是老实人"→假→A不是老实人
B说"A说真话"→假→B不是老实人
A是骗子或普通人，B是骗子或普通人

B说"A说真话"→假→B也是骗子？但只有1个骗子

如果A是普通人→A有时真有时假→"我是老实人"为假✓
B说"A说真话"→假→B是骗子

**答案：C是老实人，B是骗子，A是普通人**

### 练习 10：综合
> 5 位同事周一到周五各值班一天：
> (1) 王不在周一和周五值班
> (2) 李在张的第二天值班
> (3) 赵在周三值班
> (4) 刘在王的前一天值班
> 问：王星期几值班？

<details>
<summary>点击查看答案</summary>

**答案：周四**

解析：
由(3)：赵=周三
由(2)：李在张后一天 → 张,李 相邻
由(4)：刘在王前一天 → 刘,王 相邻
由(1)：王≠周一，王≠周五

赵=周三，剩余位置1,2,4,5

张李相邻，刘王相邻
4个位置放2对相邻的

位置1,2：张李 或 刘王
位置4,5：另一对

如果张李在1,2：张=1,李=2 或 张=2,李=1
刘王在4,5：刘=4,王=5 → 但王≠周五 ✗

所以刘王不能在4,5的(4,5)位置

如果刘=4,王=5 → 王=周五 ✗
如果刘=1,王=2 → 王=周二 ✓
张李在4,5：张=4,李=5 或 张=5,李=4（不相邻 ✗）

等等，4和5是相邻的，张=4,李=5 ✓

方案：刘=1,王=2,赵=3,张=4,李=5
检查：(1)王≠周一✓，王≠周五✓ (2)李在张后一天✓ (3)赵=周三✓ (4)刘在王前一天✓

但王=周二？让我再检查...

另一种：张李在1,2，刘王在4,5
张=1,李=2,赵=3,刘=4,王=5 → 王=周五 ✗

所以唯一方案：刘=1,王=2,赵=3,张=4,李=5

**答案：王在周二值班**

</details>

## 十、本节核心结论（必背）

1. **综合推理必须写下来**，不要在脑子里转
2. **排序题**：列位置，标条件，从严入手
3. **分组题**：画框架，标限制，先处理"必须/不能"
4. **匹配题**：画表格，用排除法逐步填
5. **真假话**：找矛盾对，假设验证
6. **连锁推理**：串联条件，肯前推后，否后推前
7. **矩阵型**：排除法，从限制最多的格子入手
8. **时间控制**：单题不超过 3 分钟，做不出先跳
9. **"不一定相邻"** ≠ "相邻"，注意区分
10. **填 ✓ 后立即排除**同行同列的其他位置
11. **"一定"需要唯一解**，"可能"只需存在一种情况
