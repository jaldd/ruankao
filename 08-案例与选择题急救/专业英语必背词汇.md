# 软考高项专业英语必背词汇

> **考情速览**：综合知识科目第71-75题，共5道英文选择题，每题1分。考的是**项目管理专业术语的英文缩写和全称**，不考语法、不考写作、不考阅读理解。**"熟悉术语，看懂题干，5分必拿"**。
>
> **备考策略**：不需要拼写，看到英文能对应中文即可。建议花1-2小时通读2遍，重点记忆英文与中文的对应关系。

## 七、易混淆术语辨析

| 术语A | 术语B | 区别 |
|-------|-------|------|
| Validate Scope（确认范围） | Verify Scope（核实范围） | Validate=客户验收可交付成果；Verify=检查是否符合需求 |
| Quality Assurance（质量保证） | Quality Control（质量控制） | QA关注过程，QC关注产品 |
| Issue（问题） | Risk（风险） | Issue=已发生的问题；Risk=潜在的不确定性 |
| Assumption（假设） | Constraint（制约因素） | Assumption=假设为真；Constraint=确定的限制 |
| Contingency Reserve（应急储备） | Management Reserve（管理储备） | 应急储备应对"已知-未知"，管理储备应对"未知-未知" |
| Preventive Action（预防措施） | Corrective Action（纠正措施） | 预防=防患于未然；纠正=纠偏已发生的偏差 |
| Precision（精确度） | Accuracy（准确度） | Precision=重复测量一致性；Accuracy=与真实值接近度 |
| Resource Leveling（资源平衡） | Resource Smoothing（资源平滑） | 平衡可能改变关键路径；平滑不改变关键路径 |
| Project Life Cycle（项目生命周期） | Product Life Cycle（产品生命周期） | 项目周期含启动到收尾；产品周期含概念到退市 |

## 九、答题技巧

1. **先看选项再读题**：选项中常有中文提示，先扫选项可快速判断考什么知识点
2. **抓关键词**：题干中的专业术语是解题核心，如看到"baseline"想"基准"，看到"variance"想"偏差"
3. **排除法**：排除与题干主题明显无关或表述过于绝对的选项
4. **上下文推测**：不认识的单词结合语境推测，项目管理英语题语境很窄
5. **常见题干句式**：
   - "Which of the following best defines..." → 定义类
   - "The primary objective of...is to..." → 目的类
   - "In which situation would...be most appropriate?" → 应用类
   - "What is the formula for..." → 公式类
