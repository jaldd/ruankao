# 08 二叉树

## 核心思路

二叉树题的核心是 **递归思维**：把问题拆解为"根节点做什么 + 左子树 + 右子树"。

**遍历框架**：
```python
def traverse(root):
    if not root:
        return
    # 前序位置
    traverse(root.left)
    # 中序位置
    traverse(root.right)
    # 后序位置
```

关键理解：
- **前序**：刚进入节点时执行，适合自顶向下传递信息
- **中序**：左子树处理完后执行，BST中序=有序序列
- **后序**：左右子树都处理完后执行，适合自底向上汇总信息
