## 四、每周学习流程（与 AI 协作的标准动作）

每周与 AI 学习时，按以下流程推进：

```
1. 打开 00-每周计划/README.md，确认本周任务
2. 选择一个具体知识点（如"均值不等式"或"假言命题"）
3. 对 AI 说："今天学 [具体知识点]，请讲概念 + 给 3 道例题 + 1 道练习题"
4. 学完后，让 AI 把核心内容整理成 Markdown，你贴到对应文件里
5. 做错的题贴到对应科目的 91_错题本.md
6. 周末复盘：让 AI 根据本周错题，出 10 道变式题检测
```

## 五、不报班，靠 AI 自学，可行性分析

**结论：完全可行，但有三个前提。**

详细分析见 `99-工具与资源/02_AI自学可行性分析.md`，这里给结论：

| 科目 | AI 能替代培训班吗？ | 关键补充 |
|------|------------------|----------|
| 数学 | 能（讲解 + 出题 + 改错） | 必须自己动笔算，不能只看 |
| 逻辑 | 非常能（AI 强于大部分培训老师） | 形式逻辑是 AI 的舒适区 |
| 写作 | 非常能（批改 + 范文 + 改稿） | 需要自己先动笔，AI 再改 |
| 英语 | 能（阅读讲题 + 作文批改 + 翻译） | 单词必须自己背，AI 替不了 |
| 复试 | 能（模拟面试 + 问题准备 + 自我介绍修改） | 最终要对着镜子练 |

**AI 替代不了的三件事**：
1. **限时模考环境**：必须自己买答题卡，严格限时做真题
2. **报名信息差**：报名时间、现场确认、准考证打印，必须自己盯官网
3. **持续自律**：没人催你上课，你能不能每天学 2 小时

如果你自律 OK，AI 自学不仅可行，而且比培训班更高效（随时问、随时练、针对性强）。
