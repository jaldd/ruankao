## 一、排列组合

### 1. 两个基本原理

**加法原理**（分类）：完成一件事有 n 类方法，各类分别有 m₁, m₂, ..., mₙ 种，共 m₁+m₂+...+mₙ 种

**乘法原理**（分步）：完成一件事分 n 步，各步分别有 m₁, m₂, ..., mₙ 种，共 m₁×m₂×...×mₙ 种

> **口诀**：分类用加法，分步用乘法

**判断关键**：做完一步后，这件事是否完成了？
- 完成了 → 分类 → 加法
- 没完成 → 分步 → 乘法

**例**：从 A 到 B 有 3 条路，从 B 到 C 有 4 条路，从 A 到 C（必须经过 B）有几种走法？
- 分步：3 × 4 = 12

**例**：从 A 到 B 有 3 条火车线路，2 条汽车线路，有几种走法？
- 分类：3 + 2 = 5

### 2. 排列与组合

**排列**（有序）：
```
Aₙᵐ = n!/(n-m)! = n(n-1)(n-2)...(n-m+1)
```

**组合**（无序）：
```
Cₙᵐ = n!/[m!(n-m)!] = Aₙᵐ/m!
```

**组合的重要性质**：
```
Cₙᵐ = Cₙⁿ⁻ᵐ          （互补性，选 m 个等于不选 n-m 个）
Cₙ⁰ = Cₙⁿ = 1         （边界值）
Cₙᵐ = Cₙ₋₁ᵐ⁻¹ + Cₙ₋₁ᵐ （杨辉三角递推）
```

**常见组合数速算**：
```
Cₙ⁰ = 1,  Cₙ¹ = n,  Cₙ² = n(n-1)/2
C₅² = 10,  C₆² = 15,  C₆³ = 20,  C₈³ = 56
C₁₀³ = 120,  C₁₀⁴ = 210
```

**排列与组合的区别**：
- 排列：选出来**还要排顺序** → Aₙᵐ
- 组合：选出来**不排顺序** → Cₙᵐ
- 判断方法：交换两个元素的位置，结果是否不同？不同→排列，相同→组合

### 3. 常见题型与解法（必背套路）

| 题型 | 解法 | 关键词 |
|------|------|--------|
| 捆绑法 | 先捆成一个整体，再排列 | "必须相邻" |
| 插空法 | 先排其他，再插空 | "不能相邻" |
| 隔板法 | n 个相同物品分 m 组 | "相同物品分组" |
| 特殊元素优先 | 先安排特殊元素/位置 | "某人必须/不能" |
| 间接法 | 总数 - 不满足的 | "至少" |
| 涂色问题 | 按颜色种数分类讨论 | "涂色/染色" |
| 分组分配 | 先分组后分配 | "分成几组/分给几人" |
| 错排问题 | Dₙ = n!(1-1/1!+1/2!-...+(-1)ⁿ/n!) | "都不在原位" |

### 4. 捆绑法详解

> 题目特征：某几个元素**必须相邻**

**步骤**：
1. 将相邻元素"捆绑"为一个整体
2. 将整体与其他元素排列
3. 整体内部排列
4. 总数 = 整体间排列 × 整体内排列

**例**：5 人排一排，A、B 必须相邻，有几种排法？
- 捆绑 AB 为一个整体：4 个元素排列 = A₄⁴ = 24
- AB 内部排列 = A₂² = 2
- 总数 = 24 × 2 = 48

**多个捆绑组**：5 人排一排，A、B 相邻，C、D 也相邻，有几种排法？
- AB 捆绑、CD 捆绑：3 个元素排列 = A₃³ = 6
- AB 内部 = 2，CD 内部 = 2
- 总数 = 6 × 2 × 2 = 24

### 5. 插空法详解

> 题目特征：某几个元素**不能相邻**

**步骤**：
1. 先排没有限制的元素
2. 将有限制的元素插入空隙中
3. 空隙数 = 已排元素数 + 1

**例**：5 人排一排，A、B 不相邻，有几种排法？
- 先排其他 3 人：A₃³ = 6
- 3 人形成 4 个空隙，A、B 插入：A₄² = 12
- 总数 = 6 × 12 = 72

**注意**：如果 A、B 也不相邻且 C、D 也不相邻
- 先排无限制的 1 人：1 种
- 1 人形成 2 个空隙，先插 A、B：A₂² = 2
- 现在有 3 人形成 4 个空隙，插 C、D：A₄² = 12
- 总数 = 1 × 2 × 12 = 24

### 6. 隔板法详解

> 题目特征：**相同物品**分给**不同对象**，每人至少 1 个

**公式**：n 个相同物品分给 m 个人，每人至少 1 个，有 Cₙ₋₁ᵐ⁻¹ 种

**原理**：n 个物品排一排，中间有 n-1 个间隔，插入 m-1 块隔板，分成 m 组

**例**：10 个相同苹果分给 3 个人，每人至少 1 个，有几种分法？
- C₁₀₋₁³⁻¹ = C₉² = 36

**变式 1**：每人至少 2 个？
- 先每人分 1 个，剩 7 个，每人至少 1 个
- C₇₋₁³⁻¹ = C₆² = 15

**变式 2**：允许有人分到 0 个？
- 先每人借 1 个（保证至少 1 个），共需 n+m 个
- Cₙ₊ₘ₋₁ᵐ⁻¹
- 例：10 个苹果分给 3 人，允许 0 个：C₁₂² = 66

**变式 3**：每人分到的数量有上限？
- 间接法：总数 - 超过上限的情况

### 7. 分组分配问题（重点难点）

**核心区分**：
- **分组**：只关心"谁和谁一组"，不区分组的编号
- **分配**：关心"哪组分到哪个位置"，区分组的编号

**分组规则**：
- 均分组要除以组数的阶乘（因为组之间不可区分）
- 非均分组不除
- 部分均分，除以均分组数的阶乘

**例 1**：6 人分成 3 组，每组 2 人，有几种分法？
- C₆² × C₄² × C₂² / 3! = 15 × 6 × 1 / 6 = 15
- 除以 3! 是因为 3 组人数相同，组之间不可区分

**例 2**：6 人分成 3 组，人数分别为 1, 2, 3，有几种分法？
- C₆¹ × C₅² × C₃³ = 6 × 10 × 1 = 60
- 不除，因为 3 组人数不同，天然可区分

**例 3**：6 人分成 3 组，人数分别为 2, 2, 2，再分配到 3 个不同房间，有几种？
- 先分组：C₆² × C₄² × C₂² / 3! = 15
- 再分配：15 × A₃³ = 15 × 6 = 90

**例 4**：9 人分成 3 组，人数分别为 2, 3, 4，再分配到 3 个不同房间，有几种？
- 先分组：C₉² × C₇³ × C₄⁴ = 36 × 35 × 1 = 1260
- 再分配：1260 × A₃³ = 1260 × 6 = 7560

**例 5**：6 人分成 3 组，人数分别为 1, 1, 4，有几种分法？
- C₆¹ × C₅¹ × C₄⁴ / 2! = 6 × 5 × 1 / 2 = 15
- 除以 2! 是因为 2 个 1 人组人数相同

### 8. 涂色问题

> 题目特征：给图形的各区域涂色，相邻区域颜色不同

**方法**：按颜色种数分类讨论

**例**：用 5 种颜色给 4 个区域涂色，相邻区域不同色，有几种？

若 4 个区域排成一排，相邻不同色：
- 第 1 区域：5 种
- 第 2 区域：4 种（与第 1 不同）
- 第 3 区域：4 种（与第 2 不同）
- 第 4 区域：4 种（与第 3 不同）
- 总数 = 5 × 4 × 4 × 4 = 320

**环形涂色**（首尾也相邻）：
- 用递推法或分类讨论
- 设 f(n) 为 n 个区域环形涂色的方法数
- f(n) = (k-1)ⁿ + (-1)ⁿ × (k-1)，其中 k 为颜色数

### 9. 错排问题

> 题目特征：n 个元素都不在原来的位置

**公式**：
```
D₁ = 0
D₂ = 1
D₃ = 2
D₄ = 9
D₅ = 44
Dₙ = (n-1)(Dₙ₋₁ + Dₙ₋₂)
```

**例**：4 封信全部装错信封，有几种装法？
- D₄ = 9

**例**：5 人互换座位，都不坐原位，有几种？
- D₅ = 44

## 二、概率

### 1. 古典概型

**条件**：
1. 有限个基本事件
2. 每个基本事件等可能

**公式**：
```
P(A) = A 包含的基本事件数 / 基本事件总数
```

**解题步骤**：
1. 确定总的基本事件数（分母）
2. 确定有利事件数（分子）
3. 相除得概率

### 2. 条件概率

```
P(A|B) = P(AB) / P(B)
```

**含义**：在 B 已经发生的条件下，A 发生的概率

**乘法公式**：
```
P(AB) = P(A) × P(B|A) = P(B) × P(A|B)
```

**例**：袋中有 3 红 2 白，不放回取 2 次，第 2 次取红球的概率？
- 方法 1（全概率）：P = 3/5 × 2/4 + 2/5 × 3/4 = 6/20 + 6/20 = 3/5
- 方法 2（抽签模型）：与顺序无关，第 2 次取红概率 = 红球数/总数 = 3/5

### 3. 独立事件

**定义**：P(AB) = P(A) · P(B)，即 A 的发生不影响 B 的概率

**判断方法**：P(B|A) = P(B)

**独立重复试验（伯努利概型）**：
```
n 次独立试验中，事件 A 恰好发生 k 次的概率：
P = Cₙᵏ · pᵏ · (1-p)ⁿ⁻ᵏ
```

**例**：某产品合格率 0.9，取 3 件，恰有 2 件合格的概率？
- P = C₃² × 0.9² × 0.1¹ = 3 × 0.81 × 0.1 = 0.243

**"至少一次"型**：
```
P(至少一次) = 1 - P(一次都没有) = 1 - (1-p)ⁿ
```

**例**：每次命中率 0.3，投 5 次，至少命中 1 次的概率？
- P = 1 - 0.7⁵ = 1 - 0.16807 = 0.83193

### 4. 互斥事件与对立事件

**互斥**：A、B 不能同时发生 → P(A∪B) = P(A) + P(B)

**对立**：A、B 互斥且必有一个发生 → P(Ā) = 1 - P(A)

> **技巧**："至少一次"型问题，用对立事件：
> P(至少一次) = 1 - P(一次都没有)

**互斥 vs 独立**：
- 互斥：P(AB) = 0（不可能同时发生）
- 独立：P(AB) = P(A)P(B)（互不影响）
- 若 P(A)>0 且 P(B)>0，互斥与独立**不能同时成立**

### 5. 常见概率模型

| 模型 | 公式 | 特点 |
|------|------|------|
| 摸球模型 | C(好)ᵏ × C(差)ⁿ⁻ᵏ / C(总)ⁿ | 不放回取球 |
| 抽签模型 | 概率与抽签顺序无关 | 不放回，等概率 |
| 伯努利模型 | Cₙᵏpᵏ(1-p)ⁿ⁻ᵏ | 独立重复试验 |
| 几何概型 | 长度/面积/体积之比 | 无限等可能 |

### 6. 摸球模型详解

> 从含 a 个好球、b 个差球的袋中取 n 个，恰有 k 个好球的概率

```
P = Cₐᵏ × C_bⁿ⁻ᵏ / Cₐ₊_bⁿ
```

**例**：袋中有 5 红 3 白 8 个球，取 3 个，恰有 2 红 1 白的概率？
- P = C₅² × C₃¹ / C₈³ = 10 × 3 / 56 = 30/56 = 15/28

**变式**：至少 1 个红球？
- P = 1 - P(全是白) = 1 - C₃³/C₈³ = 1 - 1/56 = 55/56

### 7. 抽签模型详解

> n 个人抽签，m 个好签，第 k 个人抽到好签的概率

**结论**：概率 = m/n，与 k 无关！

**证明**（以 3 人 2 好签为例）：
- 第 1 人抽到好签概率 = 2/3
- 第 2 人抽到好签概率 = 1/3 × 2/2 + 2/3 × 1/2 = 2/6 + 2/6 = 2/3
- 第 3 人同理也是 2/3

> **考试技巧**：遇到"第几次取到"的问题，直接用总数比

### 8. 全概率公式

```
P(A) = P(A|B₁)P(B₁) + P(A|B₂)P(B₂) + ... + P(A|Bₙ)P(Bₙ)
```

其中 B₁, B₂, ..., Bₙ 是样本空间的一个划分

**例**：甲乙丙三车间生产同一产品，产量比 2:3:5，次品率分别为 5%, 4%, 2%。任取一件，是次品的概率？
- P = 0.2×0.05 + 0.3×0.04 + 0.5×0.02 = 0.01 + 0.012 + 0.01 = 0.032

## 三、数据描述

### 1. 平均数

**简单平均数**：
```
x̄ = (x₁ + x₂ + ... + xₙ) / n
```

**加权平均数**：
```
x̄ = (f₁x₁ + f₂x₂ + ... + fₙxₙ) / (f₁ + f₂ + ... + fₙ)
```

**性质**：
- ∑(xᵢ - x̄) = 0（偏差之和为零）
- x̄ 是使 ∑(xᵢ - a)² 最小的 a 值

### 2. 方差与标准差

**方差**：
```
s² = [(x₁-x̄)² + (x₂-x̄)² + ... + (xₙ-x̄)²] / n
```

**简化计算**：
```
s² = (x₁² + x₂² + ... + xₙ²)/n - x̄²
```

**标准差**：s = √s²

**重要性质**（考试常考）：
| 变换 | 平均数 | 方差 |
|------|--------|------|
| 每个数据加 a | x̄ + a | s²（不变） |
| 每个数据乘 k | kx̄ | k²s² |
| 每个数据变为 kx + a | kx̄ + a | k²s² |

**例**：数据 2, 4, 6, 8, 10 的方差为 s²，则数据 3xᵢ - 1 的方差为？
- 新方差 = 9s²（乘以 3 的平方，减 1 不影响方差）

### 3. 频数与频率

- **频数**：某组数据的个数
- **频率** = 频数 / 总数
- 频率之和 = 1
- 频数之和 = 总数

### 4. 频数分布直方图

**读图要点**：
- 横轴：数据分组
- 纵轴：频率/组距（注意不是频率！）
- 每个小矩形的面积 = 频率 = (频率/组距) × 组距
- 所有小矩形面积之和 = 1

**例**：某直方图纵轴为频率/组距，某组组距为 10，纵坐标为 0.02，则该组频率 = 0.02 × 10 = 0.2

### 5. 中位数与众数

- **中位数**：数据从小到大排列，中间的数（n 为奇数取中间，n 为偶数取中间两个的平均）
- **众数**：出现次数最多的数

**例**：数据 2, 3, 3, 5, 7, 8, 9
- 中位数 = 5
- 众数 = 3

## 四、例题精讲

### 例 1：捆绑法

> 3 男 3 女排一排，3 个女生必须相邻，有几种排法？

**解析**：
3 女捆绑为 1 个整体：4 个元素排列 = A₄⁴ = 24
3 女内部排列 = A₃³ = 6
总数 = 24 × 6 = 144

**答案：144**

---

### 例 2：插空法

> 3 男 3 女排一排，3 个女生互不相邻，有几种排法？

**解析**：
先排 3 男：A₃³ = 6
3 男形成 4 个空隙，3 女插入：A₄³ = 24
总数 = 6 × 24 = 144

**答案：144**

---

### 例 3：隔板法

> 将 12 个相同的小球放入 4 个不同的盒子，每个盒子至少 2 个，有几种放法？

**解析**：
先每个盒子放 1 个，剩 8 个，每个盒子至少 1 个
C₈₋₁⁴⁻¹ = C₇³ = 35

**答案：35**

---

### 例 4：隔板法变式

> 将 20 个相同的小球放入 4 个不同的盒子，每个盒子至少 3 个，有几种放法？

**解析**：
先每个盒子放 2 个（保证至少 3 个还差 1 个），剩 20-8=12 个
现在每个盒子至少 1 个即可
C₁₂₋₁⁴⁻¹ = C₁₁³ = 165

**答案：165**

---

### 例 5：古典概型

> 从 1, 2, 3, 4, 5 中任取 2 个数，两数之和为偶数的概率？

**解析**：
总取法：C₅² = 10

两数之和为偶数：两奇或两偶
- 两奇：C₃² = 3（1,3,5 中取 2 个）
- 两偶：C₂² = 1（2,4 中取 2 个）
- 共 4 种

P = 4/10 = 2/5

**答案：2/5**

---

### 例 6：古典概型（骰子）

> 同时掷两枚骰子，两数之和为 8 的概率？

**解析**：
总事件数 = 6 × 6 = 36

和为 8 的情况：(2,6), (3,5), (4,4), (5,3), (6,2) 共 5 种

P = 5/36

**答案：5/36**

---

### 例 7：伯努利概型

> 某人投篮命中率 0.6，投 5 次，至少命中 3 次的概率？

**解析**：
P(至少3次) = P(3次) + P(4次) + P(5次)

= C₅³(0.6)³(0.4)² + C₅⁴(0.6)⁴(0.4)¹ + C₅⁵(0.6)⁵(0.4)⁰

= 10 × 0.216 × 0.16 + 5 × 0.1296 × 0.4 + 1 × 0.07776

= 0.3456 + 0.2592 + 0.07776

= 0.68256

**答案：约 0.683**

---

### 例 8：方差性质

> 数据 2, 4, 6, 8, 10 的方差为 s²，若每个数据乘以 3 再加 1，新数据的方差为？

**解析**：
x̄ = (2+4+6+8+10)/5 = 6

s² = [(2-6)²+(4-6)²+(6-6)²+(8-6)²+(10-6)²]/5
= [16+4+0+4+16]/5 = 8

新数据 = 3xᵢ + 1，新方差 = 9s² = 72

**答案：72**

---

### 例 9：分组分配

> 将 6 本不同的书分给甲乙丙三人，每人 2 本，有几种分法？

**解析**：
这是分配问题（人不同，即组可区分）
C₆² × C₄² × C₂² = 15 × 6 × 1 = 90

注意：这里不除以 3!，因为甲乙丙是不同的人，组天然可区分

**答案：90**

---

### 例 10：分组分配（部分均分）

> 将 6 本不同的书分成 3 组，一组 1 本，一组 1 本，一组 4 本，有几种分法？

**解析**：
C₆¹ × C₅¹ × C₄⁴ / 2! = 6 × 5 × 1 / 2 = 15

除以 2! 是因为两个 1 本的组人数相同，组之间不可区分

**答案：15**

---

### 例 11：特殊元素优先法

> 0, 1, 2, 3, 4 五个数字能组成多少个无重复数字的三位偶数？

**解析**：
偶数末位为 0, 2, 4

**末位为 0**：首位从 1,2,3,4 选 = 4，第二位从剩下 3 个选 = 3
→ 4 × 3 = 12

**末位为 2 或 4**：末位 2 种，首位不能为 0，从剩下 3 个非零数选 = 3，第二位从剩下 3 个选 = 3
→ 2 × 3 × 3 = 18

总数 = 12 + 18 = 30

**答案：30**

---

### 例 12：涂色问题

> 用 4 种颜色给正方形的 4 个区域涂色，相邻区域不同色，有几种？

**解析**：

## 六、易错点分析

### 易错点 1：排列与组合混淆

❌ **错误**：从 10 人中选 3 人站一排，有 C₁₀³ = 120 种
✅ **正确**：站一排有顺序，应用 A₁₀³ = 720 种

> **判断方法**：交换两个人的位置，结果不同→排列，相同→组合

### 易错点 2：捆绑法忘记内部排列

❌ **错误**：AB 必须相邻，5 人排列 = A₄⁴ = 24 种
✅ **正确**：A₄⁴ × A₂² = 24 × 2 = 48 种

> **提醒**：捆绑后别忘了内部排列！

### 易错点 3：隔板法"至少"条件搞错

❌ **错误**：10 个苹果分 3 人，每人至少 2 个，C₉² = 36
✅ **正确**：先每分 1 个剩 7 个，C₆² = 15

> **提醒**：隔板法公式 Cₙ₋₁ᵐ⁻¹ 要求每人至少 1 个，若至少 k 个，先分 k-1 个

### 易错点 4：分组问题忘记除以均分组阶乘

❌ **错误**：6 人分 3 组每组 2 人 = C₆²C₄²C₂² = 90
✅ **正确**：C₆²C₄²C₂² / 3! = 15

> **提醒**：均分组要除以组数的阶乘！非均分不除！

### 易错点 5：分组与分配混淆

- 分组：不区分组的编号 → 均分要除阶乘
- 分配：区分组的编号（分给不同的人/房间）→ 不除

> **关键**：看"组"是否可区分。人不同→组可区分→不除

### 易错点 6：独立与互斥混淆

❌ **错误**：A、B 独立，则 P(A∪B) = P(A) + P(B)
✅ **正确**：独立时 P(A∪B) = P(A) + P(B) - P(A)P(B)

> **提醒**：P(A∪B) = P(A) + P(B) 是互斥的公式，不是独立的

### 易错点 7：概率的"至少"问题不用间接法

❌ **错误**：至少 1 次命中 = P(1次) + P(2次) + ...（计算量大易错）
✅ **正确**：P(至少1次) = 1 - P(0次) = 1 - (1-p)ⁿ

> **提醒**："至少"型概率问题，优先考虑对立事件

### 易错点 8：方差性质中加减搞混

❌ **错误**：数据变为 2x + 3，方差变为 4s² + 3
✅ **正确**：方差变为 4s²（加常数不影响方差）

> **提醒**：方差只受乘法影响，不受加法影响

### 易错点 9：0 不能做首位

❌ **错误**：用 0,1,2,3 组三位数 = A₄³ = 24
✅ **正确**：首位不能为 0，3 × 3 × 2 = 18

> **提醒**：排数字问题，0 不能做首位！

### 易错点 10：环形排列忘记减 1

❌ **错误**：5 人围坐 = 5! = 120
✅ **正确**：环形排列 = (5-1)! = 24

> **提醒**：环形排列 = (n-1)!，因为旋转后相同

### 易错点 11：重复排列忘记除以重复元素的阶乘

❌ **错误**：AAABB 排列 = 5! = 120
✅ **正确**：5!/(3!×2!) = 10

> **提醒**：有重复元素时，排列数 = n!/(各重复元素的阶乘之积)

### 易错点 12：概率加法忘记减交集

❌ **错误**：P(A或B) = P(A) + P(B)
✅ **正确**：P(A∪B) = P(A) + P(B) - P(A∩B)

> **提醒**：只有互斥时 P(A∪B) = P(A) + P(B)

## 八、本节核心结论（必背）

1. **排列**：Aₙᵐ = n!/(n-m)!（有序）
2. **组合**：Cₙᵐ = n!/[m!(n-m)!]（无序）
3. **捆绑法**：相邻问题，先捆后排（别忘了内部排列！）
4. **插空法**：不相邻问题，先排后插
5. **隔板法**：相同物品分组，Cₙ₋₁ᵐ⁻¹（每人至少1个）
6. **分组**：均分除阶乘，非均分不除
7. **分配**：分组后再全排列
8. **古典概率**：P(A) = 有利事件数/总事件数
9. **伯努利**：P(k次) = Cₙᵏpᵏ(1-p)ⁿ⁻ᵏ
10. **至少一次**：1 - (1-p)ⁿ
11. **方差性质**：kx+a 的方差 = k² × 原方差
12. **环形排列**：(n-1)!
13. **错排**：D₄=9, D₅=44
