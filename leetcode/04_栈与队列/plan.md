# 04 栈与队列

## 核心思路

1. **栈**：后进先出，适用于括号匹配、表达式求值、单调栈
2. **单调栈**：维护递增/递减序列，核心场景——找下一个更大/更小元素
3. **优先队列（堆）**：Top-K问题、合并有序列表、中位数

**单调栈模板**（找下一个更大元素）：
```python
stack = []
result = [-1] * len(nums)
for i in range(len(nums)):
    while stack and nums[stack[-1]] < nums[i]:
        result[stack.pop()] = nums[i]
    stack.append(i)
```
