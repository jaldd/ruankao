## 一、平面几何

### 1. 三角形

**面积公式**：
```
S = ½ah = ½ab·sinC = √[p(p-a)(p-b)(p-c)]（海伦公式，p=(a+b+c)/2）
```

**内角和**：180°

**重要定理**：

| 定理 | 内容 |
|------|------|
| 勾股定理 | a²+b²=c²（直角三角形） |
| 正弦定理 | a/sinA=b/sinB=c/sinC=2R |
| 余弦定理 | c²=a²+b²-2ab·cosC |
| 角平分线定理 | 角平分线分对边之比=邻边之比 |

**特殊三角形**（必背）：

| 类型 | 性质 |
|------|------|
| 等边三角形 | 三边相等，三内角60°，面积=√3/4·a²，高=√3/2·a |
| 等腰直角三角形 | 两直角边相等，斜边=直角边×√2 |
| 30-60-90 | 三边比 1:√3:2 |
| 直角三角形 | 斜边中线=斜边一半 |

**相似三角形**：
- 判定：AA / SAS / SSS
- 性质：对应边成比例，面积比=相似比的平方

**中位线**：三角形中位线平行第三边且等于第三边一半

#### 例题

**例1**：△ABC中，AB=5, AC=8, ∠A=60°，求面积

> **答案：10√3**
> S=½×5×8×sin60°=20×√3/2=10√3

**例2**：直角三角形两直角边为3和4，求斜边上的高

> **答案：12/5**
> 斜边=5，S=½×3×4=6=½×5×h → h=12/5

**例3**：等边三角形面积为9√3，求边长

> **答案：6**
> √3/4·a²=9√3 → a²=36 → a=6

**例4**：条件充分性判断：△ABC是直角三角形。
(1) 三边之比为3:4:5
(2) 三边之比为5:12:13

> **答案：D**
> (1)：3²+4²=5² ✓。充分。
> (2)：5²+12²=13² ✓。充分。

**例5**：△ABC中，D是BC中点，则△ABD与△ADC的面积之比为？

> **答案：1:1**
> 同高，底BD=DC，面积相等。

**例6**：条件充分性判断：△ABC的面积大于10。
(1) AB=5, AC=6, ∠A=60°
(2) AB=5, AC=6, ∠A=90°

> **答案：B**
> (1)：S=½×5×6×√3/2=15√3/2≈13>10 ✓。充分。
> 等等，15√3/2≈12.99>10 ✓。充分。
> (2)：S=½×5×6=15>10 ✓。充分。
> **答案：D**

**例7**：两个相似三角形的面积比为4:9，则它们的周长比为？

> **答案：2:3**
> 面积比=相似比的平方 → 相似比=2:3 → 周长比=2:3

### 2. 四边形

**平行四边形**：对边平行且相等，面积=底×高，对角线互相平分

**矩形**：面积=长×宽，对角线=√(长²+宽²)

**菱形**：四边相等，面积=½×d₁×d₂=底×高，对角线互相垂直平分

**正方形**：面积=a²，对角线=a√2

**梯形**：面积=½×(上底+下底)×高=中位线×高，中位线=½×(上底+下底)

#### 例题

**例1**：菱形两条对角线长分别为6和8，求面积和边长

> **答案：面积24，边长5**
> 面积=½×6×8=24
> 边长=√(3²+4²)=5

**例2**：矩形对角线长为10，面积为48，求长和宽

> **答案：8和6**
> a²+b²=100, ab=48
> (a+b)²=100+96=196 → a+b=14
> (a-b)²=100-96=4 → a-b=2
> a=8, b=6

**例3**：条件充分性判断：四边形ABCD是菱形。
(1) AB=BC=CD=DA
(2) 对角线AC⊥BD

> **答案：A**
> (1)：四边相等 → 菱形 ✓。充分。
> (2)：对角线垂直可能是筝形，不一定是菱形。不充分。
> 等等，对角线互相垂直平分才是菱形。只说垂直不够。不充分。
> **答案：A**

**例4**：梯形上底4，下底10，高6，求面积和中位线

> **答案：面积42，中位线7**
> 中位线=(4+10)/2=7
> 面积=7×6=42

**例5**：条件充分性判断：平行四边形ABCD的面积为24。
(1) AB=6，高为4
(2) 对角线长分别为6和8，且对角线互相垂直

> **答案：D**
> (1)：面积=6×4=24 ✓。充分。
> (2)：对角线互相垂直的平行四边形是菱形，面积=½×6×8=24 ✓。充分。

### 3. 圆

**基本公式**：
```
周长 C=2πr=πd
面积 S=πr²
弧长 l=nπr/180
扇形面积 S=nπr²/360=½lr
```

**重要性质**：
- 直径所对圆周角=90°
- 同弧圆周角相等
- 圆心角=2×圆周角（同弧）
- 切线垂直于过切点的半径

**阴影面积求法**（常考）：
1. 割补法
2. 差值法：大-小
3. 对称法

#### 例题

**例1**：圆的面积为25π，求周长

> **答案：10π**
> πr²=25π → r=5 → C=10π

**例2**：扇形半径为6，圆心角120°，求弧长和面积

> 弧长=120×π×6/180=4π
> 面积=120×π×36/360=12π
> **答案：弧长4π，面积12π**

**例3**：条件充分性判断：圆的面积大于9π。
(1) 圆的周长大于6π
(2) 圆的半径大于3

> **答案：D**
> (1)：C>6π → r>3 → S>9π ✓。充分。
> (2)：r>3 → S>9π ✓。充分。

**例4**：正方形内切圆的面积与正方形面积之比为？

> **答案：π/4**
> 设正方形边长2a，内切圆半径a
> 圆面积=πa²，正方形面积=4a²
> 比值=π/4

**例5**：半圆中内接一个等腰直角三角形，斜边在直径上，求三角形面积与半圆面积之比

> 设半径r，斜边=2r，直角边=2r/√2=r√2
> 三角形面积=½×(r√2)²=r²
> 半圆面积=½πr²
> 比值=r²/(½πr²)=2/π
> **答案：2/π**

#### 练习

<details>
<summary>练习1：圆环内圆半径3，外圆半径5，求圆环面积</summary>

π×5²-π×3²=25π-9π=16π
**答案：16π**

</details>

<details>
<summary>练习2：条件充分性判断：扇形面积等于圆面积的三分之一。(1) 圆心角120° (2) 弧长等于半径的2π/3倍</summary>

(1)：120/360=1/3 ✓。充分。
(2)：弧长=2πr/3=nπr/180 → n=120° → 1/3 ✓。充分。
**答案：D**

</details>

## 二、解析几何

### 1. 直线

**斜率**：k=(y₂-y₁)/(x₂-x₁)

**直线方程**：

| 形式 | 方程 | 适用条件 |
|------|------|----------|
| 点斜式 | y-y₀=k(x-x₀) | k存在 |
| 斜截式 | y=kx+b | k存在 |
| 两点式 | (y-y₁)/(y₂-y₁)=(x-x₁)/(x₂-x₁) | x₁≠x₂,y₁≠y₂ |
| 截距式 | x/a+y/b=1 | a≠0,b≠0 |
| 一般式 | Ax+By+C=0 | — |

**两条直线位置关系**：

| 关系 | 条件 |
|------|------|
| 平行 | k₁=k₂, b₁≠b₂ |
| 重合 | k₁=k₂, b₁=b₂ |
| 相交 | k₁≠k₂ |
| 垂直 | k₁·k₂=-1 |

**点到直线距离**（必背）：
```
d = |Ax₀+By₀+C| / √(A²+B²)
```

**两条平行线距离**：
```
d = |C₁-C₂| / √(A²+B²)
```

#### 例题

**例1**：点(1,2)到直线3x+4y-5=0的距离

> **答案：6/5**
> d=|3+8-5|/√(9+16)=6/5

**例2**：过点(2,1)且与直线x-y+1=0垂直的直线方程

> **答案：x+y-3=0**
> 原直线斜率k=1，垂直斜率k'=-1
> y-1=-1(x-2) → x+y-3=0

**例3**：条件充分性判断：直线l₁与l₂平行。
(1) l₁: 2x+3y-1=0, l₂: 4x+6y+5=0
(2) l₁: x+y=1, l₂: 2x+2y=3

> **答案：D**
> (1)：l₁斜率-2/3，l₂斜率-4/6=-2/3，截距不同 → 平行 ✓。充分。
> (2)：l₁斜率-1，l₂斜率-1，截距不同 → 平行 ✓。充分。

**例4**：两条平行线2x-y+1=0与2x-y-5=0的距离

> **答案：6/√5**
> d=|1-(-5)|/√(4+1)=6/√5=6√5/5

**例5**：条件充分性判断：点(1,1)在直线l上。
(1) l: x+y=2
(2) l: 2x-y=1

> **答案：D**
> (1)：1+1=2 ✓。充分。
> (2)：2-1=1 ✓。充分。

**例6**：求过点(1,2)且与两点A(0,0), B(3,1)等距的直线方程

> AB中点(3/2, 1/2)，AB斜率1/3
> 等距直线过AB中点且与AB垂直，或过(1,2)且与AB平行
> 情况1：过AB中点且垂直AB
> 垂直斜率=-3，y-1/2=-3(x-3/2) → 3x+y-5=0
> 验证(1,2)：3+2-5=0 ✓
> 情况2：过(1,2)且平行AB
> 斜率1/3，y-2=1/3(x-1) → x-3y+5=0
> 验证：A到直线距离=|0-0+5|/√10=5/√10
> B到直线距离=|3-3+5|/√10=5/√10 ✓
> **答案：3x+y-5=0 或 x-3y+5=0**

**答案：2x-y-8=0**

### 2. 圆

**标准方程**：(x-a)²+(y-b)²=r²，圆心(a,b)，半径r

**一般方程**：x²+y²+Dx+Ey+F=0
- 圆心(-D/2, -E/2)
- 半径r=½√(D²+E²-4F)

**直线与圆的位置关系**：

| 关系 | 条件 | 交点数 |
|------|------|--------|
| 相离 | d>r | 0 |
| 相切 | d=r | 1 |
| 相交 | d<r | 2 |

**圆与圆的位置关系**：

| 关系 | 条件 |
|------|------|
| 外离 | d>r₁+r₂ |
| 外切 | d=r₁+r₂ |
| 相交 | |r₁-r₂|<d<r₁+r₂ |
| 内切 | d=|r₁-r₂| |
| 内含 | d<|r₁-r₂| |

#### 例题

**例1**：圆x²+y²-2x+4y-4=0的圆心和半径

> **答案：圆心(1,-2)，半径3**
> (x-1)²+(y+2)²=9

**例2**：判断直线x+y-1=0与圆x²+y²=2的位置关系

> **答案：相交**
> 圆心(0,0)，r=√2
> d=|0+0-1|/√2=1/√2=√2/2<√2 → 相交

**例3**：条件充分性判断：直线x+y=r与圆x²+y²=r²相切。
(1) r=1
(2) r=√2

> **答案：D**
> d=|0+0-r|/√2=r/√2
> 相切：r/√2=r → 1/√2=1，不成立。
> 等等，直线是x+y=r，圆是x²+y²=r²
> d=|0+0-r|/√2=r/√2
> 相切：d=r → r/√2=r → r=0（舍去）
> 所以任何r>0都不相切！
> **答案：E**（两个条件都不充分）

**例4**：过圆x²+y²=4上一点(√3,1)的切线方程

> **答案：√3x+y=4**
> 过圆上点(x₀,y₀)的切线：x₀x+y₀y=r²
> √3x+y=4

**例5**：条件充分性判断：圆C₁与圆C₂相切。
(1) C₁: x²+y²=1, C₂: (x-3)²+y²=4
(2) C₁: x²+y²=1, C₂: (x-2)²+y²=1

> **答案：B**
> (1)：圆心距d=3，r₁+r₂=3 → 外切 ✓。充分。
> 等等，d=3, r₁=1, r₂=2, r₁+r₂=3=d → 外切 ✓。
> (2)：圆心距d=2，r₁=1, r₂=1, r₁+r₂=2=d → 外切 ✓。
> **答案：D**

**例6**：圆x²+y²-4x+2y+1=0关于直线x-y=0对称的圆的方程

> 原圆心(2,-1)，半径=√(4+1-1)=2
> 关于y=x对称：(2,-1)→(-1,2)
> 对称圆：(x+1)²+(y-2)²=4
> **答案：(x+1)²+(y-2)²=4**

### 3. 对称问题

**点关于点对称**：P(x,y)关于(a,b)的对称点为(2a-x, 2b-y)

**点关于直线对称**：
- 关于x轴：(x,-y)
- 关于y轴：(-x,y)
- 关于y=x：(y,x)
- 关于y=-x：(-y,-x)
- 关于直线Ax+By+C=0：中点在直线上+连线与直线垂直

#### 例题

**例1**：点(2,3)关于点(1,1)的对称点

> **答案：(0,-1)**
> (2×1-2, 2×1-3)=(0,-1)

**例2**：点(2,3)关于直线y=x的对称点

> **答案：(3,2)**

**例3**：点(1,2)关于直线x+y=0的对称点

> **答案：(-2,-1)**
> 关于y=-x对称：(-y,-x)=(-2,-1)

**例4**：条件充分性判断：点(a,b)关于直线x-y+1=0的对称点在x轴上。
(1) a=b-1
(2) a=0, b=1

> 对称点求法：设对称点(a',b')
> 中点((a+a')/2,(b+b')/2)在直线上：(a+a')/2-(b+b')/2+1=0
> 连线与直线垂直：斜率×1=-1 → (b'-b)/(a'-a)×1=-1 → b'-b=-(a'-a)
> 由第二个方程：a'-a=-(b'-b) → a'=a-(b'-b)
> 对称点在x轴上：b'=0
> a'=a-(-b)=a+b, b'=0
> 代入第一个方程：(a+a+b)/2-(b+0)/2+1=0 → (2a+b)/2-b/2+1=0 → a+1=0 → a=-1
> (1)：a=b-1，不知道a=-1。不充分。
> (2)：a=0≠-1。不充分。联合：a=0,b=1，a≠-1。不充分。
> **答案：E**

## 三、立体几何

### 1. 长方体

```
体积 V=abc
表面积 S=2(ab+bc+ac)
体对角线=√(a²+b²+c²)
```

### 2. 圆柱

```
体积 V=πr²h
侧面积=2πrh
全面积=2πr(r+h)
```

### 3. 球体

```
体积 V=4πr³/3
表面积 S=4πr²
```

**球与长方体**：外接球直径=体对角线，r=½√(a²+b²+c²)

#### 例题

**例1**：长方体三个相邻面面积分别为6,8,12，求体积和对角线

> **答案：体积24，对角线√29**
> ab=6, bc=8, ac=12 → (abc)²=576 → abc=24
> a=3, b=2, c=4
> 对角线=√(9+4+16)=√29

**例2**：球的表面积为36π，求体积

> **答案：36π**
> 4πr²=36π → r=3
> V=4π×27/3=36π

**例3**：条件充分性判断：长方体的体对角线长为5。

**4**. 条件充分性判断：正方体内接于球，球的体积为32√3π/3。

(2) 正方体体对角线为2√3

<details><summary>答案</summary>

(2)：体对角线=2√3 → r=√3 → V=4√3π。不充分。
等等，正方体内接于球，球是外接球。

V=4π(a√3/2)³/3=4π×3√3a³/8/3=√3πa³/2
要V=32√3π/3 → √3πa³/2=32√3π/3 → a³=64/3...a不是整数。
让我重新算：V=32√3π/3 → 4πr³/3=32√3π/3 → r³=8√3 → r=2∛√3...这不对。
32√3π/3 = 4πr³/3 → r³=8√3 → r=2×3^(1/6)
(1)：a=2, r=√3, r³=3√3≠8√3。不充分。
(2)：2r=2√3, r=√3, 同上。不充分。

**5**. 在圆x²+y²=4上求一点P，使P到直线x+y-4=0的距离最短，并求最短距离

## 六、本节核心结论（必背）

1. **点到直线距离**：d=|Ax₀+By₀+C|/√(A²+B²)
2. **圆的标准方程**：(x-a)²+(y-b)²=r²
3. **直线与圆**：d<r相交，d=r相切，d>r相离
4. **三角形面积**：S=½ab·sinC
5. **菱形面积**：½×d₁×d₂
6. **长方体外接球**：2r=体对角线=√(a²+b²+c²)
7. **阴影面积**：大减小，割补法
8. **过圆上点切线**：x₀x+y₀y=r²
9. **相似三角形**：面积比=相似比的平方
10. **对称问题**：关于y=x互换坐标，关于原点取反
