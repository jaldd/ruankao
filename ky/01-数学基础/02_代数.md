## 一、整式与分式

### 1. 乘法公式（必背）

| 公式 | 展开 |
|------|------|
| (a+b)² | a² + 2ab + b² |
| (a-b)² | a² - 2ab + b² |
| (a+b+c)² | a² + b² + c² + 2ab + 2bc + 2ac |
| a² - b² | (a+b)(a-b) |
| a³ + b³ | (a+b)(a² - ab + b²) |
| a³ - b³ | (a-b)(a² + ab + b²) |
| (a+b)³ | a³ + 3a²b + 3ab² + b³ |
| (a-b)³ | a³ - 3a²b + 3ab² - b³ |

**常用推论**：
- a² + b² = (a+b)² - 2ab = (a-b)² + 2ab
- (a+b)² + (a-b)² = 2(a²+b²)
- (a+b)² - (a-b)² = 4ab

### 2. 因式分解

**常用方法**：
1. **提取公因式**：ma + mb = m(a+b)
2. **公式法**：a²-b² = (a+b)(a-b)
3. **十字相乘法**：x²+(p+q)x+pq = (x+p)(x+q)
4. **分组分解法**：ax+ay+bx+by = (a+b)(x+y)

**常考结论**：
- 若 x²+px+q = (x+a)(x+b)，则 p=a+b，q=ab
- 因式分解要彻底

### 3. 分式

**基本性质**：A/B = (A×C)/(B×C) = (A÷C)/(B÷C)（C≠0）

**分式有意义的条件**：分母≠0
**分式值为0的条件**：分子=0 且分母≠0

#### 例题

**例1**：若 x² - 3x + 2 = 0，则 x³ - 3x² + 2x + 1 = ?

> **答案：1**
> x³-3x²+2x = x(x²-3x+2) = x·0 = 0
> 所以 x³-3x²+2x+1 = 1

**例2**：已知 a+b = 5，ab = 6，则 a²+b² = ?

> **答案：13**
> a²+b² = (a+b)²-2ab = 25-12 = 13

**例3**：条件充分性判断：a³+b³ = 8。
(1) a+b = 2
(2) ab = -3

> **答案：C**
> a³+b³ = (a+b)(a²-ab+b²) = (a+b)[(a+b)²-3ab]
> (1) 单独：不知道 ab，不充分。
> (2) 单独：不知道 a+b，不充分。
> 联合：a³+b³ = 2×(4+9) = 26 ≠ 8。联合也不充分？
> 等等，(a+b)²-3ab = 4-3×(-3) = 4+9 = 13。a³+b³ = 2×13 = 26。
> **答案：E**（联合也不充分）

**例4**：化简 (x²-4)/(x²-4x+4) ÷ (x+2)/(x-2)

> **答案：1**
> = (x-2)(x+2)/(x-2)² × (x-2)/(x+2) = 1

**例5**：条件充分性判断：分式 (x²-1)/(x-1) 的值为正整数。
(1) x = 2
(2) x > 1 且 x 为整数

> **答案：A**
> (x²-1)/(x-1) = (x+1)(x-1)/(x-1) = x+1（x≠1）
> (1)：x=2 → x+1=3，正整数✓。充分。
> (2)：x>1 且为整数 → x=2,3,4,... → x+1=3,4,5,...都是正整数✓。充分。
> **答案：D**

## 二、函数

### 1. 一元二次函数

**一般式**：f(x) = ax²+bx+c（a≠0）
**顶点式**：f(x) = a(x-h)²+k，顶点(h,k)
**顶点坐标**：(-b/2a, (4ac-b²)/4a)
**对称轴**：x = -b/2a

**判别式**：Δ = b²-4ac

| Δ | 图像与x轴 | 根的个数 |
|---|----------|---------|
| Δ>0 | 两个交点 | 两个不等实根 |
| Δ=0 | 一个交点 | 两个相等实根 |
| Δ<0 | 无交点 | 无实根 |

**最值**：a>0 有最小值，a<0 有最大值，都在 x=-b/2a 处取到

**二次函数与不等式**：
- a>0：f(x)>0 解集在两根之外，f(x)<0 解集在两根之间
- a<0：反过来

#### 例题

**例1**：f(x) = -x²+4x+5 在 [0,3] 上的最大值和最小值？

> **答案：最大值9，最小值5**
> f(x) = -(x-2)²+9，对称轴 x=2 在 [0,3] 内
> f(2)=9（最大），f(0)=5，f(3)=8 → 最小值5

**例2**：条件充分性判断：二次函数 f(x) = x²-2x+a 与 x 轴有两个交点。
(1) a < 0
(2) a < 1

> **答案：D**
> Δ = 4-4a > 0 → a < 1
> (1)：a<0 < 1 ✓。充分。
> (2)：a<1 ✓。充分。

**例3**：若 f(x) = x²-2ax+a+2 在 (-∞,1] 上递减，则 a 的取值范围是？

> **答案：a ≥ 1**
> 对称轴 x = a，要在 (-∞,1] 上递减 → a ≥ 1（对称轴在1的右边或等于1）

**例4**：函数 f(x) = x²+bx+c 满足 f(1)=0, f(2)=3，求 f(3)=?

> **答案：8**
> f(1)=1+b+c=0 → b+c=-1
> f(2)=4+2b+c=3 → 2b+c=-1
> 解得 b=0, c=-1
> f(x)=x²-1, f(3)=8

**例5**：条件充分性判断：f(x) = (a-1)x²+2x+1 的图像是抛物线。
(1) a ≠ 1
(2) a > 1

> **答案：A**
> 抛物线要求 a-1≠0 → a≠1
> (1)：a≠1 ✓。充分。
> (2)：a>1 → a≠1 ✓。充分。但 a<1 且 a≠1 也行，所以(2)是充分但不是必要条件。
> **答案：D**（两个条件都充分）

### 2. 指数函数

**y = aˣ**（a>0, a≠1）
- a>1 单调递增，0<a<1 单调递减
- 恒过(0,1)，值域(0,+∞)

**运算法则**：
- aˣ·aʸ = aˣ⁺ʸ
- (aˣ)ʸ = aˣʸ
- a⁻ˣ = 1/aˣ
- (ab)ˣ = aˣbˣ

#### 例题

**例1**：2^(x+1) = 8，则 x = ?

> **答案：2**
> 2^(x+1) = 2³ → x+1=3 → x=2

**例2**：条件充分性判断：aˣ > aʸ。
(1) a > 1, x > y
(2) 0 < a < 1, x < y

> **答案：D**
> (1)：a>1 单调递增，x>y → aˣ>aʸ ✓。充分。
> (2)：0<a<1 单调递减，x<y → aˣ>aʸ ✓。充分。

### 3. 对数函数

**y = logₐx**（a>0, a≠1）
- a>1 单调递增，0<a<1 单调递减
- 恒过(1,0)，定义域(0,+∞)，值域R

**运算法则**：
```
logₐ(MN) = logₐM + logₐN
logₐ(M/N) = logₐM - logₐN
logₐMⁿ = n·logₐM
换底公式：logₐb = lnb/lna
```

**重要推论**：logₐa=1, logₐ1=0, a^(logₐN)=N

#### 例题

**例1**：log₂8 + log₃9 = ?

> **答案：5**
> log₂8 = 3, log₃9 = 2 → 3+2=5

**例2**：若 log₂x + log₂y = 3，则 x+y 的最小值为？

> **答案：4√2**
> log₂(xy)=3 → xy=8
> x+y ≥ 2√(xy) = 2√8 = 4√2

**例3**：条件充分性判断：logₐ2 > logₐ3。
(1) 0 < a < 1
(2) a > 1

> **答案：A**
> (1)：0<a<1 单调递减，2<3 → logₐ2>logₐ3 ✓。充分。
> (2)：a>1 单调递增，2<3 → logₐ2<logₐ3 ✗。不充分。

**例4**：计算 log₂3 · log₃4 · log₄8

> **答案：3**
> 换底：log₂3 · (log₂4/log₂3) · (log₂8/log₂4) = log₂8 = 3

**例5**：条件充分性判断：lgx + lgy = 2。
(1) x = 10, y = 10
(2) xy = 100

> **答案：D**
> lgx+lgy = lg(xy) = 2 → xy=100
> (1)：10×10=100 ✓。充分。
> (2)：xy=100 ✓。充分。

## 三、方程

### 1. 一元二次方程

**标准形式**：ax²+bx+c=0（a≠0）
**求根公式**：x = (-b±√Δ)/2a

**韦达定理**（必考！）：
```
x₁+x₂ = -b/a
x₁·x₂ = c/a
```

**韦达定理常见变形**（必背）：
```
x₁²+x₂² = (x₁+x₂)²-2x₁x₂
1/x₁+1/x₂ = (x₁+x₂)/(x₁x₂)
|x₁-x₂| = √((x₁+x₂)²-4x₁x₂) = √Δ/|a|
x₁³+x₂³ = (x₁+x₂)³-3x₁x₂(x₁+x₂)
(x₁-x₂)² = (x₁+x₂)²-4x₁x₂
```

**根的分布**（常考）：

| 条件 | 等价条件 |
|------|----------|
| 两个正根 | Δ≥0, x₁+x₂>0, x₁x₂>0 |
| 两个负根 | Δ≥0, x₁+x₂<0, x₁x₂>0 |
| 一正一负 | x₁x₂<0（即c/a<0） |
| 两根都大于k | Δ≥0, x₁+x₂>2k, f(k)>0 |
| 一根大于k一根小于k | f(k)<0 |

#### 例题

**例1**：x₁,x₂ 是方程 x²-3x+1=0 的两根，求 x₁²+x₂²

> **答案：7**
> x₁+x₂=3, x₁x₂=1
> x₁²+x₂² = 9-2 = 7

**例2**：x₁,x₂ 是方程 2x²-6x+3=0 的两根，求 1/x₁+1/x₂ 和 |x₁-x₂|

> 1/x₁+1/x₂ = (x₁+x₂)/(x₁x₂) = 3/(3/2) = 2
> |x₁-x₂| = √Δ/|a| = √(36-24)/2 = √12/2 = √3
> **答案：2 和 √3**

**例3**：条件充分性判断：方程 x²+mx+n=0 的两根之差的绝对值为 4。
(1) m=4, n=0
(2) m=2, n=-3

> **答案：D**
> |x₁-x₂| = √(m²-4n)
> (1)：√(16-0) = 4 ✓。充分。
> (2)：√(4+12) = √16 = 4 ✓。充分。

**例4**：方程 x²-2(m-1)x+m²-1=0 有两个不相等的正根，求 m 的范围

> 条件：Δ>0, x₁+x₂>0, x₁x₂>0
> Δ = 4(m-1)²-4(m²-1) = 8(1-m) > 0 → m<1
> x₁+x₂ = 2(m-1) > 0 → m>1
> 矛盾！**答案：不存在这样的m**

**例5**：条件充分性判断：方程 x²+2(k-1)x+k²-1=0 有两个实根。
(1) k > 1
(2) 0 < k < 1

> Δ = 4(k-1)²-4(k²-1) = 4(k²-2k+1-k²+1) = 4(2-2k) = 8(1-k) ≥ 0 → k ≤ 1
> (1)：k>1 → Δ<0 ✗。不充分。
> (2)：0<k<1 → Δ>0 ✓。充分。
> **答案：B**

**例6**：若方程 x²-(k-2)x+k²+3k+5=0 的两根为 α,β，则 α²+β² 的最大值是？

> α²+β² = (α+β)²-2αβ = (k-2)²-2(k²+3k+5)
> = k²-4k+4-2k²-6k-10 = -k²-10k-6
> = -(k²+10k+6) = -(k+5)²+19
> 最大值为 19（当 k=-5 时取到）
> 但需要验证 Δ≥0：Δ = (k-2)²-4(k²+3k+5) = k²-4k+4-4k²-12k-20 = -3k²-16k-16
> Δ = -3k²-16k-16 ≥ 0 → 3k²+16k+16 ≤ 0
> 解 3k²+16k+16=0：k = (-16±√(256-192))/6 = (-16±8)/6
> k = -4/3 或 k = -4
> 所以 -4 ≤ k ≤ -4/3
> k=-5 不在范围内！需要重新求最大值。
> f(k) = -k²-10k-6 在 [-4, -4/3] 上的最大值：
> 对称轴 k=-5 在区间左边，f(k) 在 [-4, -4/3] 上递减
> f(-4) = -16+40-6 = 18
> f(-4/3) = -16/9+40/3-6 = -16/9+120/9-54/9 = 50/9 ≈ 5.56
> **答案：18**

## 四、不等式

### 1. 一元二次不等式

**口诀**：a>0 时，大于取两边，小于取中间

**解法步骤**：
1. 化标准形式
2. 求对应方程的根
3. 根据 a 正负画数轴

### 2. 均值不等式（必考重点）

**基本形式**：
```
a+b ≥ 2√(ab)（a,b>0）
a+b+c ≥ 3∛(abc)（a,b,c>0）
```

**求最值三步法**：
1. 确认各项为正
2. 判断"和定"还是"积定"
3. 验证取等条件

**常见凑配技巧**：
```
y = x + 1/x ≥ 2（x>0）
y = x + 4/x ≥ 4（x>0）
y = 2x + 8/x ≥ 8（x>0）
y = x(4-x) ≤ 4（0<x<4）
```

### 3. 绝对值不等式

```
|x| < a ⟺ -a < x < a（a>0）
|x| > a ⟺ x<-a 或 x>a（a>0）
|a|-|b| ≤ |a±b| ≤ |a|+|b|
```

#### 例题

**例1**：解不等式 x²-5x+6<0

> **答案：2<x<3**
> x²-5x+6=(x-2)(x-3)<0 → 2<x<3

**例2**：条件充分性判断：不等式 x²+ax+b<0 的解集为 (1,2)。
(1) a=-3, b=2
(2) a+b=-1

> **答案：A**
> 解集(1,2) → x²+ax+b=(x-1)(x-2)=x²-3x+2 → a=-3, b=2
> (1)：a=-3, b=2 ✓。充分。
> (2)：a+b=-1，多解（如a=0,b=-1不满足）。不充分。

**例3**：x>1，求 y=x+4/(x-1) 的最小值

> **答案：5**
> 令 t=x-1>0，y=t+1+4/t = t+4/t+1 ≥ 2√4+1 = 5
> 取等 t=2, x=3

**例4**：条件充分性判断：x²-2x-3>0。
(1) x>3
(2) x<-1

> **答案：D**
> x²-2x-3=(x-3)(x+1)>0 → x>3 或 x<-1
> (1)：x>3 ✓。充分。
> (2)：x<-1 ✓。充分。

**例5**：0<x<π/2，求 y=sinx+1/sinx 的最小值

> **答案：2**
> sinx>0, y=sinx+1/sinx ≥ 2√(sinx·1/sinx) = 2
> 取等 sinx=1, x=π/2。但 x<π/2，取不到等号！
> 实际上 sinx 在 (0,π/2) 上趋近于1但不等于1，y趋近于2但不等于2。
> **答案：y>2，无最小值**（但考试一般选2，注意边界问题）

**答案：[-1,4]**

</details>

## 五、数列

### 1. 等差数列

**通项公式**：aₙ = a₁+(n-1)d

**前n项和**：Sₙ = n(a₁+aₙ)/2 = na₁+n(n-1)d/2

**重要性质**（必背）：
```
m+n = p+q → aₘ+aₙ = aₚ+aq
aₙ = Sₙ-Sₙ₋₁（n≥2）
S₂ₙ₋₁ = (2n-1)·aₙ
Sₙ, S₂ₙ-Sₙ, S₃ₙ-S₂ₙ 成等差数列
等差数列中奇数项和与偶数项和的关系
```

### 2. 等比数列

**通项公式**：aₙ = a₁·qⁿ⁻¹

**前n项和**：
```
Sₙ = a₁(1-qⁿ)/(1-q)（q≠1）
Sₙ = na₁（q=1）
```

**重要性质**：
```
m+n = p+q → aₘ·aₙ = aₚ·aq
Sₙ, S₂ₙ-Sₙ, S₃ₙ-S₂ₙ 成等比数列（公比qⁿ）
```

### 3. 数列求和常用方法

| 方法 | 适用场景 |
|------|----------|
| 公式法 | 等差/等比 |
| 分组求和 | 可拆成等差+等比 |
| 裂项相消 | 1/n(n+1) = 1/n-1/(n+1) |
| 错位相减 | 等差×等比 |

**常见裂项**：
```
1/n(n+1) = 1/n - 1/(n+1)
1/n(n+2) = 1/2·(1/n - 1/(n+2))
1/(√n+√(n+1)) = √(n+1) - √n
```

#### 例题

**例1**：等差数列 {aₙ} 中，a₂+a₅+a₈=30，则 S₉=?

> **答案：90**
> a₂+a₅+a₈ = 3a₅ = 30 → a₅=10
> S₉ = 9a₅ = 90

**例2**：等比数列 {aₙ} 中，a₁=1, a₄=8，求 S₆

> **答案：63**
> q³=8 → q=2
> S₆ = (1-2⁶)/(1-2) = (1-64)/(-1) = 63

**例3**：条件充分性判断：数列 {aₙ} 是等差数列。
(1) aₙ = 3n+1
(2) Sₙ = 2n²+n

> **答案：D**
> (1)：aₙ=3n+1，aₙ₊₁-aₙ=3（常数）✓。充分。
> (2)：Sₙ=2n²+n，aₙ=Sₙ-Sₙ₋₁=2n²+n-2(n-1)²-(n-1)=4n-1，公差4 ✓。充分。

**例4**：等差数列 {aₙ} 中，S₁₀=100, S₂₀=400，求 S₃₀

> **答案：900**
> S₁₀, S₂₀-S₁₀, S₃₀-S₂₀ 成等差数列
> 即 100, 300, S₃₀-400 成等差
> 公差 d=200，S₃₀-400=500 → S₃₀=900

**例5**：条件充分性判断：等比数列 {aₙ} 的公比 q>1。
(1) aₙ>0 且 aₙ₊₁>aₙ
(2) a₁>0 且 S₃>S₂

> **答案：A**
> (1)：aₙ>0, aₙ₊₁>aₙ → q=aₙ₊₁/aₙ>1 ✓。充分。
> (2)：a₁>0, S₃>S₂ → a₃>0 → a₁q²>0。S₃-S₂=a₃=a₁q²>0，但q可以是0<q<1（a₁>0, q=0.5, a₃>0）。不充分。
> 等等，S₃>S₂ → a₃>0，这只说明q≠0，不能确定q>1。不充分。
> **答案：A**

**例6**：求和 S = 1×2 + 2×4 + 3×8 + ... + n×2ⁿ

> **答案：(n-1)·2ⁿ⁺¹+2**
> 错位相减法：
> S = 1×2 + 2×4 + 3×8 + ... + n×2ⁿ
> 2S = 1×4 + 2×8 + ... + (n-1)×2ⁿ + n×2ⁿ⁺¹
> 2S-S = -2-4-8-...-2ⁿ+n×2ⁿ⁺¹
> S = n×2ⁿ⁺¹ - (2+4+8+...+2ⁿ)
> S = n×2ⁿ⁺¹ - 2(2ⁿ-1) = n×2ⁿ⁺¹ - 2ⁿ⁺¹+2 = (n-1)·2ⁿ⁺¹+2

**例7**：条件充分性判断：Sₙ = 1+1/2+1/3+...+1/n > 3。
(1) n ≥ 11
(2) n ≥ 10

> S₁₀ ≈ 2.93, S₁₁ ≈ 3.02
> (1)：n≥11 → Sₙ≥S₁₁>3 ✓。充分。
> (2)：n≥10 → S₁₀≈2.93<3。不充分。
> **答案：A**

**1**. 已知 a+b=6, ab=5，求 a³+b³

<details><summary>答案</summary>
a³+b³=(a+b)³-3ab(a+b)=216-3×5×6=216-90=126

**2**. 条件充分性判断：二次方程 x²+2x+a=0 至少有一个负根。
(1) a > 0
(2) a < 1

<details><summary>答案</summary>
x₁+x₂=-2<0，两根之和为负。
若两根都为正：x₁+x₂>0，矛盾。所以至少有一个负根（实际上总有至少一个负根）。
任何条件都充分。
**答案：D**（无条件也成立，但题目给了条件，两个条件各自都充分）
</details>

**3**. 等差数列 {aₙ} 中，S₄=1, S₈=4，求 a₁₇+a₁₈+a₁₉+a₂₀

<details><summary>答案</summary>
S₄=1, S₈-S₄=3, S₁₂-S₈, S₁₆-S₁₂, S₂₀-S₁₆ 成等差，公差为2。
S₁₂-S₈=5, S₁₆-S₁₂=7, S₂₀-S₁₆=9
a₁₇+a₁₈+a₁₉+a₂₀ = S₂₀-S₁₆ = 9
**答案：9**
</details>

**4**. 条件充分性判断：f(x) = (a-2)x²+2(a-2)x-4 的值恒为负。
(1) -2 < a < 2
(2) a = 0

<details><summary>答案</summary>
恒为负条件：a-2<0 且 Δ<0
a-2<0→a<2
Δ=4(a-2)²+16(a-2)=4(a-2)(a-2+4)=4(a-2)(a+2)<0→-2<a<2
综合：-2<a<2
(1)-2<a<2✓。(2)a=0∈(-2,2)✓。
**答案：D**
</details>

**5**. 求和 1/(1×3) + 1/(3×5) + 1/(5×7) + ... + 1/((2n-1)(2n+1))

</details>
