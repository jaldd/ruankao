# 00 排序与查找（Day 0 前置）

> 这是整个刷题计划的前置基础，正式开始 Day 1 之前先搞定这些。

## 一图看懂6大排序

| 算法 | 平均时间 | 最坏时间 | 空间 | 稳定性 | 核心思想 |
|------|---------|---------|------|--------|---------|
| 冒泡排序 | O(n²) | O(n²) | O(1) | ✅ 稳定 | 相邻比较，大的往后冒 |
| 选择排序 | O(n²) | O(n²) | O(1) | ❌ 不稳定 | 每轮选最小的放前面 |
| 插入排序 | O(n²) | O(n²) | O(1) | ✅ 稳定 | 扑克牌插入，往前找位置 |
| 快速排序 | O(nlogn) | O(n²) | O(logn) | ❌ 不稳定 | 选基准，小的左大的右，递归 |
| 归并排序 | O(nlogn) | O(nlogn) | O(n) | ✅ 稳定 | 拆两半分别排，再合并 |
| 堆排序 | O(nlogn) | O(nlogn) | O(1) | ❌ 不稳定 | 建大顶堆，逐个取堆顶 |

## 学习建议

1. **先理解思想**：每个算法看一遍图解动画（推荐 visualgo.net）
2. **再背模板**：每个算法的 Java 模板代码在同目录下，直接背
3. **手写一遍**：关掉代码，自己从头写，写不出就再看，直到能默写
4. **对比记忆**：快排 vs 归并、冒泡 vs 插入，找异同点

## 查找算法

二分查找是查找的核心，模板在 BinarySearch.java 中，包含：
- 标准二分（找精确值）
- 找左边界（第一个 >= target）
- 找右边界（最后一个 <= target）
