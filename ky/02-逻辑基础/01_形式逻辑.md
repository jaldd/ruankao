## 一、假言命题（形式逻辑的重中之重）

### 1. 三种标准形式

| 自然语言 | 逻辑形式 | 符号化 | 口诀 |
|----------|----------|--------|------|
| 如果 A，那么 B | A → B | A → B | 前推后 |
| 只有 A，才 B | B → A | B → A | 后推前 |
| 除非 A，否则 B | 非 A → B | ¬A → B | 否前推后 |

**记忆口诀**：
- "如果…那么…" → **前推后**
- "只有…才…" → **后推前**
- "除非…否则…" → **否前推后**（或理解为：非 B → A）

> **注意**："除非 A，否则 B" 等价于 "如果非 A，那么 B"，即 ¬A → B。
> 也等价于 "A 或 B"。

### 2. 更多关键词转换

| 关键词 | 逻辑形式 | 说明 |
|--------|----------|------|
| 如果 A，那么 B | A → B | 充分条件 |
| 只要 A，就 B | A → B | 充分条件 |
| 若 A，则 B | A → B | 充分条件 |
| 一旦 A，就 B | A → B | 充分条件 |
| A 是 B 的充分条件 | A → B | 充分条件 |
| 只有 A，才 B | B → A | 必要条件 |
| 除非 A，否则不 B | B → A | 必要条件 |
| A 是 B 的必要条件 | B → A | 必要条件 |
| A 是 B 的前提 | B → A | 必要条件 |
| A 是 B 的基础 | B → A | 必要条件 |
| 不 A，就不 B | B → A | 必要条件 |
| A 当且仅当 B | A ↔ B | 充要条件 |

### 3. 变形规则（逆否命题）

```
原命题：A → B
逆命题：B → A          （不一定成立）
否命题：¬A → ¬B        （不一定成立）
逆否命题：¬B → ¬A      （一定成立！与原命题等价）
```

**核心结论**：
> **A → B 等价于 ¬B → ¬A**
>
> 也就是说："如果 A 那么 B" = "如果非 B 那么非 A"

### 4. 推理规则（只能这么推）

已知 A → B 为真：

| 推理 | 形式 | 是否有效 | 说明 |
|------|------|----------|------|
| 肯定前件 | A → B，A 真 → B 真 | ✅ 有效 | 前真推后真 |
| 否定后件 | A → B，¬B → ¬A | ✅ 有效 | 后假推前假 |
| 否定前件 | A → B，¬A → ? | ❌ 无效 | 前假，后件不确定 |
| 肯定后件 | A → B，B → ? | ❌ 无效 | 后真，前件不确定 |

**记忆口诀**：
> **肯前必肯后，否后必否前**
> **否前不确定，肯后不确定**

### 5. 假言命题的等价形式

```
A → B 等价于以下所有形式：
1. ¬B → ¬A        （逆否命题）
2. ¬A ∨ B         （"非 A 或 B"）
3. ¬(A ∧ ¬B)      （"不可能 A 且非 B"）
```

**例**："如果下雨，那么地湿" 等价于：
- "如果地不湿，那么没下雨"
- "或者没下雨，或者地湿"
- "不可能下雨且地不湿"

### 6. 链式推理（串联）

```
A → B，B → C  ⇒  A → C
A → B，B → C，C → D  ⇒  A → D
```

**例**：
(1) 如果甲去，那么乙去
(2) 如果乙去，那么丙去
(3) 丙没去

由 (1)(2) 串联：甲 → 乙 → 丙，即 甲 → 丙
由 (3) ¬丙，否后必否前：¬甲

## 二、联言命题（且）

### 1. 基本形式

**形式**：A ∧ B（A 且 B）

**真假**：一假则假，全真才真。

| A | B | A ∧ B |
|---|---|-------|
| 真 | 真 | 真 |
| 真 | 假 | 假 |
| 假 | 真 | 假 |
| 假 | 假 | 假 |

### 2. 负判断（矛盾）

```
¬(A ∧ B) = ¬A ∨ ¬B
```

> "并非 A 且 B" = "非 A 或 非 B"

**例**：并非"小王既聪明又勤奋" = 小王或者不聪明，或者不勤奋

### 3. 常见关键词

- 既…又… / 不但…而且… / 虽然…但是… / …和… / …同时…

## 三、选言命题（或）

### 1. 相容选言

**形式**：A ∨ B（A 或 B）

**真假**：一真则真，全假才假

| A | B | A ∨ B |
|---|---|-------|
| 真 | 真 | 真 |
| 真 | 假 | 真 |
| 假 | 真 | 真 |
| 假 | 假 | 假 |

**负判断**：
```
¬(A ∨ B) = ¬A ∧ ¬B
```

**推理规则**：已知 A ∨ B 为真
- 若 A 假 → B 真（否定肯定式）
- 若 A 真 → B 不确定

### 2. 不相容选言

**形式**：A ⊕ B（要么 A，要么 B）

**真假**：一真一假为真，同真同假为假

| A | B | A ⊕ B |
|---|---|-------|
| 真 | 真 | 假 |
| 真 | 假 | 真 |
| 假 | 真 | 真 |
| 假 | 假 | 假 |

> 考试中如果没有特别说明"要么…要么…"，"或"默认是**相容选言**。

## 四、模态命题

### 1. 四种模态

| 模态 | 形式 | 含义 |
|------|------|------|
| 必然肯定 | □A | 一定 A |
| 必然否定 | □¬A | 一定不 A |
| 可能肯定 | ◇A | 可能 A |
| 可能否定 | ◇¬A | 可能不 A |

### 2. 对当关系

```
        □A（必然A）
       /    \
    反对    矛盾
     /        \
◇¬A（可能不A）——◇A（可能A）
    下反对
```

- **矛盾关系**：□A ↔ ¬◇¬A（必然 A = 不可能不 A）
- **矛盾关系**：□¬A ↔ ¬◇A（必然不 A = 不可能 A）
- **反对关系**：□A 和 □¬A 不能同真，可以同假
- **下反对关系**：◇A 和 ◇¬A 不能同假，可以同真

### 3. 模态命题的负判断

```
¬□A = ◇¬A     （并非必然 A = 可能不 A）
¬□¬A = ◇A     （并非必然不 A = 可能 A）
¬◇A = □¬A     （并非可能 A = 必然不 A）
¬◇¬A = □A     （并非可能不 A = 必然 A）
```

**例**："并非所有学生必然及格" = "有的学生可能不及格"

## 五、直言命题（性质命题）

### 1. 四种基本形式

| 名称 | 形式 | 含义 | 例子 |
|------|------|------|------|
| 全称肯定（A） | 所有 S 都是 P | SAP | 所有鸟都会飞 |
| 全称否定（E） | 所有 S 都不是 P | SEP | 所有鸟都不会飞 |
| 特称肯定（I） | 有的 S 是 P | SIP | 有的鸟会飞 |
| 特称否定（O） | 有的 S 不是 P | SOP | 有的鸟不会飞 |

### 2. 对当关系（逻辑方阵）

```
        A（所有S都是P）
       /    \
    反对    矛盾
     /        \
O（有的S不是P）——I（有的S是P）
    下反对
```

- **矛盾关系**：A 与 O 互为矛盾，E 与 I 互为矛盾
  - A 真 → O 假，O 真 → A 假
  - E 真 → I 假，I 真 → E 假
- **反对关系**：A 与 E 不能同真，可以同假
- **下反对关系**：I 与 O 不能同假，可以同真
- **差等关系**：A 真 → I 真，E 真 → O 真

### 3. 负判断

```
¬SAP = SOP    （并非所有S都是P = 有的S不是P）
¬SEP = SIP    （并非所有S都不是P = 有的S是P）
¬SIP = SEP    （并非有的S是P = 所有S都不是P）
¬SOP = SAP    （并非有的S不是P = 所有S都是P）
```

**例**："并非所有企业家都有MBA学位" = "有的企业家没有MBA学位"

## 六、三段论

### 1. 基本结构

```
大前提：所有 M 都是 P
小前提：所有 S 都是 M
结  论：所有 S 都是 P
```

### 2. 三段论规则

1. **中项至少周延一次**（中项 M 在前提中至少有一次是"所有"）
2. **前提中不周延的项，结论中也不得周延**
3. **两个否定前提不能得出结论**
4. **前提有一个否定，结论必否定**
5. **两个特称前提不能得出结论**
6. **前提有一个特称，结论必特称**

### 3. 常见错误

❌ **中项不周延**：
> 所有鸟都会飞（M→P）
> 所有蝙蝠都会飞（S→P）
> 所以蝙蝠是鸟（S→M？）

中项"会飞"在两个前提中都是谓项，都不周延，无效推理。

❌ **四概念错误**：
> 鲁迅的小说不是一天能读完的
> 《阿Q正传》是鲁迅的小说
> 所以《阿Q正传》不是一天能读完的

两个"鲁迅的小说"含义不同（集合概念 vs 非集合概念）。

## 七、关系命题

### 1. 对称性

| 关系类型 | 定义 | 例子 |
|----------|------|------|
| 对称关系 | aRb → bRa | "等于""同学""邻居" |
| 反对称关系 | aRb → ¬bRa | "大于""父亲""早于" |
| 非对称关系 | aRb，bRa 不确定 | "喜欢""认识""帮助" |

### 2. 传递性

| 关系类型 | 定义 | 例子 |
|----------|------|------|
| 传递关系 | aRb ∧ bRc → aRc | "大于""等于""早于" |
| 反传递关系 | aRb ∧ bRc → ¬aRc | "父亲""大两岁" |
| 非传递关系 | aRb ∧ bRc，aRc 不确定 | "认识""喜欢" |

**例**：甲比乙高，乙比丙高 → 甲比丙高（传递关系）
甲是乙的父亲，乙是丙的父亲 → 甲不是丙的父亲（反传递关系）

## 八、例题精讲

### 例 1：基本转换

> 将以下语句符号化：
> "如果天下雨，那么地湿"
> "只有浇水，花才开"
> "除非参加考试，否则不能毕业"

**解析**：
- 如果天下雨，那么地湿：**雨 → 湿**（前推后）
- 只有浇水，花才开：**开 → 浇**（后推前）
- 除非参加考试，否则不能毕业：**¬考试 → ¬毕业**（否前推后）

等价于：**毕业 → 考试**

---

### 例 2：逆否命题

> "如果考上清华 MEM，那么笔试要过 200 分"
> 以下哪项一定为真？
> A. 如果笔试过 200 分，那么考上清华 MEM
> B. 如果没考上清华 MEM，那么笔试没过 200 分
> C. 如果笔试没过 200 分，那么没考上清华 MEM
> D. 只有考上清华 MEM，笔试才过 200 分

**解析**：
原命题：**考上 → 过 200**

逆否命题：**¬过 200 → ¬考上**

- A：过 200 → 考上（肯后，无效）
- B：¬考上 → ¬过 200（否前，无效）
- C：¬过 200 → ¬考上（逆否命题，✅）
- D：过 200 → 考上（"只有考上，才过 200" = 后推前 = 过 200 → 考上，错误）

**答案：C**

---

### 例 3：串联推理

> 已知：
> (1) 如果 A，那么 B
> (2) 如果 B，那么 C
> (3) 非 C
> 问：能推出什么？

**解析**：
(1) A → B
(2) B → C
串联：A → B → C，即 A → C
(3) ¬C，否后必否前：¬A

**答案：非 A**

---

### 例 4：除非

> 除非明天下雨，否则我们去爬山。
> 以下哪项一定为真？
> A. 如果明天下雨，我们就不去爬山
> B. 如果我们没去爬山，那么明天下雨了
> C. 只有明天下雨，我们才不去爬山
> D. 如果明天不下雨，我们就去爬山

**解析**：
"除非 A，否则 B" = "¬A → B"
A = 明天下雨，B = 我们去爬山

原命题：**¬下雨 → 爬山**
逆否命题：**¬爬山 → 下雨**

- A：下雨 → ¬爬山（否前，无效）
- B：¬爬山 → 下雨（逆否命题，✅）
- C：¬爬山 → 下雨（"只有下雨，才不去爬山" = 后推前 = ¬爬山 → 下雨，✅）
- D：¬下雨 → 爬山（与原命题一致，✅）

**答案：B, C, D**（单选选 D，最直接）

---

### 例 5：联言与选言的负判断

> 并非"小王既聪明又勤奋"
> 等价于以下哪项？

**解析**：
"并非（聪明 ∧ 勤奋）" = "¬聪明 ∨ ¬勤奋"
即：小王或者不聪明，或者不勤奋

**答案：小王或者不聪明，或者不勤奋**

---

### 例 6：选言推理

> 甲或乙参加了比赛。已知甲没有参加，则？

**解析**：
A ∨ B 为真，¬A → B
甲没参加 → 乙参加了

**答案：乙参加了**

---

### 例 7：直言命题矛盾

> "并非所有员工都满意" 等价于？

**解析**：
¬SAP = SOP
"有的员工不满意"

**答案：有的员工不满意**

### 例 8：三段论

> 所有成功人士都是勤奋的，所有勤奋的人都是早起的，所以？

**解析**：
M→P（勤奋→早起），S→M（成功人士→勤奋）
S→P：所有成功人士都是早起的

**答案：所有成功人士都是早起的**

### 例 9：模态命题

> "不可能所有人都通过" 等价于？

**解析**：
¬◇SAP = □¬SAP = □SOP
"必然有的人不通过"

**答案：必然有的人不通过**

### 例 10：复杂串联

> 已知：
> (1) 如果甲上场，那么乙不上场
> (2) 只有丙上场，乙才不上场
> (3) 甲上场了
> 问：丙是否上场？

**解析**：
(1) 甲 → ¬乙
(2) ¬乙 → 丙（"只有丙，才¬乙" = 后推前 = ¬乙 → 丙）
(3) 甲

串联：甲 → ¬乙 → 丙
由 (3) 甲为真，肯前必肯后：¬乙，再肯前必肯后：丙

**答案：丙上场了**

### 例 11：必要条件陷阱

> "只有经过培训，才能上岗"
> 以下哪项一定为真？
> A. 经过培训了，就能上岗
> B. 没有上岗，说明没经过培训
> C. 没有经过培训，就不能上岗
> D. 上岗了，说明经过了培训

**解析**：
"只有培训，才上岗" = 上岗 → 培训

- A：培训 → 上岗（肯后，无效）
- B：¬上岗 → ¬培训（否前，无效）
- C：¬培训 → ¬上岗（逆否命题，✅）
- D：上岗 → 培训（与原命题一致，✅）

**答案：C, D**（单选选 C，因为 C 是逆否命题最标准）

---

### 例 12：二难推理

> 如果甲去，乙不去；如果甲不去，乙也不去。问：乙去不去？

**解析**：
甲 → ¬乙
¬甲 → ¬乙

无论甲去不去，乙都不去（二难推理）

**答案：乙不去**

> **二难推理标准形式**：
> A → B，¬A → B，所以 B 一定成立

---

### 例 13：复合命题负判断

> 并非"如果下雨，我就不出门" 等价于？

**解析**：
¬(A → B) = ¬(¬A ∨ B) = A ∧ ¬B

即：下雨 ∧ 出门（下雨但我还是出门了）

**答案：下雨了但我还是出门了**

> **重要**：假言命题的矛盾 = 前件真且后件假

### 例 14：关系命题

> 甲比乙高，乙比丙高，丁比甲高。以下哪项一定为真？
> A. 丁比丙高
> B. 丙比甲高
> C. 乙比丁高
> D. 甲比丁高

**解析**：
"比…高"是传递关系
丁 > 甲 > 乙 > 丙

- A：丁 > 丙（✅ 传递可得）
- B：丙 > 甲（❌）
- C：乙 > 丁（❌）
- D：甲 > 丁（❌）

**答案：A**

### 例 15：直言命题推理

> "有些大学生是党员"为真，以下哪项一定为真？
> A. 所有大学生都是党员
> B. 有些党员是大学生
> C. 有些大学生不是党员
> D. 所有党员都是大学生

**解析**：
SIP 为真
- A：SAP（不确定）
- B：换位推理 SIP → PIS，"有些党员是大学生"（✅）
- C：SOP（不确定）
- D：PAS（不确定）

**答案：B**

## 九、条件充分性判断式逻辑题

### 逻辑判断 1

> 能确定甲是否参加了比赛
>
> (1) 如果甲参加，那么乙参加
> (2) 乙没有参加

**解析**：
条件(1)：甲 → 乙
条件(2)：¬乙

联合：¬乙 → ¬甲（否后必否前），甲没参加，充分
单独都不充分

**答案：C**

---

### 逻辑判断 2

> 能确定小张是否通过了考试
>
> (1) 如果小张不努力，那么他不能通过
> (2) 小张努力了

**解析**：
条件(1)：¬努力 → ¬通过，即 通过 → 努力
条件(2)：努力

联合：努力了，但肯后不确定（努力不一定通过），不充分

**答案：E**

---

### 逻辑判断 3

> 能推出"乙参加了"
>
> (1) 甲或乙参加了
> (2) 甲没有参加

**解析**：
条件(1)：甲 ∨ 乙
条件(2)：¬甲

联合：¬甲，由甲∨乙 → 乙（否定肯定式），充分
单独都不充分

**答案：C**

---

### 逻辑判断 4

> 能推出"并非所有人都及格"
>
> (1) 张三没有及格
> (2) 有人及格了

**解析**：
条件(1)：张三没及格 → 有的人没及格 → ¬SAP = SOP，即并非所有人都及格，充分
条件(2)：有人及格 = SIP，不能推出 SOP，不充分

**答案：A**

## 十、易错点分析

### 易错点 1：混淆充分条件和必要条件

❌ **错误**："只有 A 才 B" = A → B
✅ **正确**："只有 A 才 B" = B → A

> **提醒**："只有"引出的是必要条件，放在后面推

### 易错点 2：否前/肯后推结论

❌ **错误**：A → B，¬A → ¬B
✅ **正确**：A → B，¬A 推不出 ¬B

> **提醒**：否前不确定！肯后不确定！

### 易错点 3："除非"的理解

❌ **错误**："除非 A 否则 B" = A → B
✅ **正确**："除非 A 否则 B" = ¬A → B

> **提醒**：除非 = if not，"除非 A" = "如果不 A"

### 易错点 4：假言命题的矛盾

❌ **错误**：A → B 的矛盾是 A → ¬B
✅ **正确**：A → B 的矛盾是 A ∧ ¬B（前件真且后件假）

> **提醒**：要否定一个条件句，需要前件成立但后件不成立

### 易错点 5：选言命题的推理方向

❌ **错误**：A ∨ B，A → B
✅ **正确**：A ∨ B，¬A → B（否定肯定式）

> **提醒**：选言命题只能通过否定一个来肯定另一个

### 易错点 6：全称命题与特称命题

❌ **错误**："并非所有人都及格" = "所有人都不及格"
✅ **正确**："并非所有人都及格" = "有的人不及格"

> **提醒**：¬SAP = SOP，不是 SEP

### 易错点 7：模态命题的双重否定

❌ **错误**："不可能不" = "不可能"
✅ **正确**："不可能不" = "必然"

> **提醒**：¬◇¬ = □

### 易错点 8：三段论中项不周延

❌ **错误**：
> 所有鸟都会飞
> 所有蝙蝠都会飞
> 所以蝙蝠是鸟

✅ **正确**：中项"会飞"在两个前提中都不周延，推理无效

> **提醒**：中项至少周延一次

### 易错点 9："有的"的理解

❌ **错误**："有的 A 是 B" 意味着 "有的 A 不是 B"
✅ **正确**："有的 A 是 B" 只表示至少有一个 A 是 B，不排除全部都是

> **提醒**："有的"在逻辑中 = "至少有一个"，不暗示"不是全部"

### 易错点 10：混淆矛盾关系与反对关系

❌ **错误**："所有鸟都会飞"的矛盾是"所有鸟都不会飞"
✅ **正确**："所有鸟都会飞"的矛盾是"有的鸟不会飞"

> **提醒**：矛盾关系 = 一真一假，不能同真也不能同假

## 十二、本节核心结论（必背）

1. **如果 A 那么 B** = A → B（前推后）
2. **只有 A 才 B** = B → A（后推前）
3. **除非 A 否则 B** = ¬A → B（否前推后）
4. **A → B 等价于 ¬B → ¬A**（逆否命题）
5. **肯前必肯后，否后必否前**
6. **否前不确定，肯后不确定**
7. **¬(A ∧ B) = ¬A ∨ ¬B**（德摩根律）
8. **¬(A ∨ B) = ¬A ∧ ¬B**（德摩根律）
9. **A → B 等价于 ¬A ∨ B**
10. **A → B 的矛盾是 A ∧ ¬B**
11. **¬SAP = SOP，¬SEP = SIP**（直言命题矛盾）
12. **¬□A = ◇¬A，¬◇A = □¬A**（模态命题矛盾）
13. **选言推理：A ∨ B，¬A → B**（否定肯定式）
14. **二难推理：A→B，¬A→B，则 B 必真**
15. **中项至少周延一次**（三段论规则）
