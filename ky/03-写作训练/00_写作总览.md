# 管理类联考写作总览

> 分值：65 分
> - 论证有效性分析：30 分（600 字，建议 25 分钟）
> - 论说文：35 分（700 字，建议 30-35 分钟）
> - **两篇作文总共只有 60 分钟左右，必须速战速决**

---

## 一、论证有效性分析（小作文）

### 本质
给你一篇有逻辑漏洞的材料（约 500 字），**找出漏洞并分析为什么错**。

### 常见逻辑谬误（必须会识别）

| 谬误类型 | 特征 | 示例 |
|----------|------|------|
| 概念混淆/偷换概念 | 前后说的不是一个东西 | "相对论是物理学理论，所以做人要相对地看问题" |
| 以偏概全 | 个体推整体，样本不足 | "我认识的几个大学生都找不到工作，所以大学生就业形势很差" |
| 因果不当 | A 发生在 B 前，就认为 A 导致 B | "鸡叫后太阳升起，所以鸡叫导致日出" |
| 类比不当 | 两类事物有表面相似，但本质不同 | "企业就像家庭，所以企业管理要像家长管孩子" |
| 非黑即白 | 忽视中间状态 | "不赞成就是反对" |
| 数字陷阱 | 绝对数/相对数混淆、平均数陷阱 | "某药治愈了 100 人，所以有效"（没说总人数） |
| 前提不当 | 论据本身就不成立 | "因为所有人都会死，所以张三是坏人" |

### 写作模板

```
标题：一篇有待商榷的论证 / 似是而非的论证

首段：上述材料通过一系列论证，试图得出"……"的结论。然而，该论证存在多处逻辑漏洞，具体分析如下：

中间段（写 4-5 个漏洞，每段一个）：
  首先，材料认为"……"，这一论证存在概念混淆的问题。……（分析）。因此，该结论难以成立。
  其次，……（以偏概全/因果不当/类比不当等）……
  再次，……
  此外，……

尾段：综上所述，材料在论证过程中存在概念混淆、因果不当、以偏概全等逻辑缺陷，其结论的合理性值得怀疑。若要得出可靠结论，还需补充更多论据和严谨推理。
```

## 二、论说文（大作文）

### 本质
给你一段材料（可能是一句名言、一个故事、一个现象），**提炼观点，写一篇议论文**。

### 常见主题
- 企业管理：创新、合作、诚信、风险、人才
- 个人成长：坚持、选择、担当、学习、格局
- 社会现象：工匠精神、可持续发展、数字化

### 写作结构（最稳妥：总分总）

```
标题：简短有力，体现观点（如"以创新之火，点燃发展之炬"）

第 1 段（引+立）：概括材料 → 引出观点（100 字）
第 2 段（正面论证）：讲道理 + 举例子（150 字）
第 3 段（反面论证）：不这样做的危害 + 举例（150 字）
第 4 段（怎么做）：个人/企业/社会层面（150 字）
第 5 段（结）：总结升华，呼应开头（100 字）
```

### 素材来源
你的软考高项知识就是绝佳素材！
- 风险管理 → "凡事预则立，不预则废"
- 干系人管理 → "合作共赢"
- 质量管理 → "精益求精，工匠精神"
- 领导力 → "领导者的人格魅力与团队建设"

## 三、写作训练节奏

| 阶段 | 任务 |
|------|------|
| 基础期（现在-8月） | 每周精读 2 篇范文，积累素材 |
| 强化期（9-10月） | 动笔写！小作文 10 篇，大作文 8 篇 |
| 冲刺期（11月-考前） | 限时训练，找人/AI 批改，打磨模板 |
