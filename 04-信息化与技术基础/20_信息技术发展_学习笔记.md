# 20 信息技术发展

## 一、一句话理解
信息技术发展就是"IT技术全家桶"——云计算、大数据、人工智能、物联网、区块链、5G等，知道它们是啥、能干啥、用在哪。

## 二、核心概念与框架

| 核心概念 | 说明 |
|---------|------|
| 云计算 | 通过网络提供计算资源（IaaS/PaaS/SaaS） |
| 大数据 | 海量数据的采集、存储、分析、可视化 |
| 人工智能 | 让机器像人一样思考和学习 |
| 物联网 | 万物互联，传感器+网络+应用 |
| 区块链 | 分布式账本，去中心化、不可篡改 |
| 5G | 第五代移动通信，高速率、低延迟、大连接 |

## 三、关键知识点逐个讲

### 知识点1：云计算
- **什么意思**：把计算资源（服务器、存储、数据库）放在云端，按需使用。
- **为什么重要**：企业上云是趋势，考试必考。
- **考试怎么考**：考三种服务模式、四种部署模式、云计算特征。

**详细展开：**
**三种服务模式（必考！）**：
| 模式 | 英文 | 提供什么 | 用户管什么 | 例子 |
|------|------|---------|-----------|------|
| 基础设施即服务 | IaaS | 服务器、存储、网络 | 操作系统、应用、数据 | 阿里云ECS、AWS EC2 |
| 平台即服务 | PaaS | 开发平台、中间件 | 应用、数据 | 阿里云PAI、Google App Engine |
| 软件即服务 | SaaS | 完整软件应用 | 只用软件，什么都不管 | 钉钉、企业微信、Salesforce |

**记忆技巧**：
- IaaS：给你"硬件"，你自己装系统
- PaaS：给你"平台"，你自己开发应用
- SaaS：给你"软件"，你直接用

**四种部署模式**：
| 模式 | 说明 | 适用场景 |
|------|------|---------|
| 公有云 | 第三方提供，资源共享 | 中小企业、弹性需求 |
| 私有云 | 企业自建，独享资源 | 大型企业、数据敏感 |
| 混合云 | 公有云+私有云结合 | 核心业务私有、非核心公有 |
| 社区云 | 特定社区共享 | 政府、金融等特定行业 |

**云计算核心特征（NIST定义）**：
- 按需自助服务：用户自己申请资源，不用找管理员
- 广泛的网络访问：通过网络访问，不限地点
- 资源池化：资源统一池化管理，动态分配
- 快速弹性：资源快速扩展或收缩
- 可计量服务：按使用量计费

**考试要点**：
- 云计算 vs 传统IT：云计算是"租"不是"买"，是"用多少付多少"
- 云原生：容器（Docker）、微服务、DevOps、持续交付
- 边缘计算：在靠近数据源的地方处理数据，减少延迟

### 知识点2：大数据
- **什么意思**：数据量太大，传统工具处理不了，需要用新技术处理。
- **为什么重要**：数据是新时代的石油。
- **考试怎么考**：考5V特征、大数据处理流程、大数据技术栈。

**详细展开：**
**大数据5V特征（必考！）**：
| 特征 | 英文 | 说明 |
|------|------|------|
| 大量 | Volume | 数据量大，TB/PB级别 |
| 高速 | Velocity | 数据产生和处理速度快 |
| 多样 | Variety | 数据类型多，结构化+半结构化+非结构化 |
| 价值 | Value | 数据有价值，但价值密度低 |
| 真实性 | Veracity | 数据质量参差不齐，需要清洗 |

**大数据处理流程**：
```
数据采集 → 数据存储 → 数据清洗 → 数据分析 → 数据可视化
```

**大数据技术栈**：
| 环节 | 技术 |
|------|------|
| 采集 | Flume、Kafka、Logstash |
| 存储 | HDFS、HBase、MongoDB |
| 计算 | MapReduce、Spark、Flink |
| 分析 | Hive、Pig、Impala |
| 可视化 | Tableau、ECharts、FineBI |

**考试要点**：
- 大数据 vs 传统数据库：大数据处理海量、高并发、非结构化数据
- 数据仓库 vs 数据湖：数据仓库是"结构化"的，数据湖是"原始数据"的集合
- 实时计算 vs 离线计算：实时用Flink/Storm，离线用MapReduce/Spark

### 知识点3：人工智能
- **什么意思**：让机器模拟人的智能，包括学习、推理、感知等。
- **为什么重要**：AI是新一轮科技革命的核心。
- **考试怎么考**：考AI技术分类、机器学习类型、AI应用场景。

**详细展开：**
**AI技术分类**：
| 技术 | 说明 | 例子 |
|------|------|------|
| 机器学习 | 让机器从数据中学习规律 | 推荐系统、垃圾邮件过滤 |
| 深度学习 | 多层神经网络，模拟人脑 | 图像识别、语音识别 |
| 自然语言处理 | 让机器理解人类语言 | 智能客服、机器翻译 |
| 计算机视觉 | 让机器"看懂"图像 | 人脸识别、自动驾驶 |
| 知识图谱 | 把知识组织成图谱 | 智能搜索、问答系统 |

**机器学习三种类型**：
| 类型 | 说明 | 例子 |
|------|------|------|
| 监督学习 | 有标签数据训练 | 分类、回归 |
| 无监督学习 | 无标签数据，自己找规律 | 聚类、降维 |
| 强化学习 | 通过试错学习 | AlphaGo、自动驾驶 |

**AI应用场景**：
- 智能制造：AI质检、预测性维护
- 智慧医疗：AI辅助诊断、药物研发
- 智慧金融：智能风控、量化交易
- 智慧交通：自动驾驶、交通流量预测

**考试要点**：
- AI三要素：数据、算法、算力
- 弱人工智能 vs 强人工智能：弱AI是专用（如下棋），强AI是通用（像人一样）
- 目前所有AI都是弱人工智能

### 知识点4：物联网
- **什么意思**：把各种物品通过传感器和网络连接起来，实现智能识别和管理。
- **为什么重要**：万物互联是趋势。
- **考试怎么考**：考三层架构、关键技术、应用场景。

**详细展开：**
**物联网三层架构（必考！）**：
| 层次 | 功能 | 技术 |
|------|------|------|
| 感知层 | 信息采集、识别 | 传感器、RFID、二维码、摄像头 |
| 网络层 | 数据传输 | 5G、WiFi、蓝牙、ZigBee、LoRa |
| 应用层 | 数据处理、应用服务 | 云计算、大数据、AI |

**物联网关键技术**：
- 传感器技术：温度、湿度、压力、光照等
- RFID：无线射频识别，用于物流、门禁
- 嵌入式系统：把计算能力嵌入设备
- 边缘计算：在设备端处理数据，减少传输

**物联网应用场景**：
- 智能家居：智能音箱、智能门锁、智能照明
- 智慧城市：智能路灯、智能停车、环境监测
- 工业物联网：设备监控、预测性维护、质量追溯
- 农业物联网：精准灌溉、温室监控、畜牧追踪

**考试要点**：
- 物联网 vs 互联网：物联网是"物物相连"，互联网是"人人相连"
- MQTT：物联网轻量级通信协议
- 工业物联网（IIoT）是物联网在工业领域的应用

### 知识点5：区块链
- **什么意思**：分布式账本技术，数据不可篡改、去中心化。
- **为什么重要**：数字货币、供应链金融等应用场景。
- **考试怎么考**：考区块链特征、共识机制、应用场景。

**详细展开：**
**区块链核心特征**：
| 特征 | 说明 |
|------|------|
| 去中心化 | 没有中心节点，所有节点平等 |
| 不可篡改 | 数据一旦写入，无法修改 |
| 透明性 | 所有交易对参与者可见 |
| 可追溯 | 每笔交易都有完整记录 |
| 智能合约 | 自动执行的程序代码 |

**共识机制（怎么达成一致）**：
| 机制 | 说明 | 代表 |
|------|------|------|
| 工作量证明 | 算力竞争，谁先算出谁记账 | 比特币 |
| 权益证明 | 持币越多，记账权越大 | 以太坊2.0 |
| 委托权益证明 | 投票选出记账节点 | EOS |

**区块链应用场景**：
- 数字货币：比特币、数字人民币
- 供应链金融：应收账款融资、信用传递
- 溯源：食品溯源、药品溯源
- 存证：电子合同、知识产权

**考试要点**：
- 公有链 vs 联盟链 vs 私有链：公有链完全开放，联盟链部分开放，私有链封闭
- 区块链不是比特币，比特币只是区块链的一个应用
- 区块链的"不可能三角"：去中心化、安全性、可扩展性不可兼得

### 知识点6：5G
- **什么意思**：第五代移动通信技术。
- **为什么重要**：5G是万物互联的基础设施。
- **考试怎么考**：考5G特征、三大应用场景、5G vs 4G。

**详细展开：**
**5G三大特征（必考！）**：
| 特征 | 指标 | 4G对比 |
|------|------|--------|
| 高速率 | 峰值20Gbps | 4G的100倍 |
| 低延迟 | 1ms级别 | 4G的1/10 |
| 大连接 | 100万设备/平方公里 | 4G的10倍 |

**5G三大应用场景（ITU定义）**：
| 场景 | 英文 | 说明 | 应用 |
|------|------|------|------|
| 增强移动宽带 | eMBB | 高速上网 | 4K/8K视频、VR/AR |
| 超可靠低延迟通信 | uRLLC | 低延迟高可靠 | 自动驾驶、远程手术 |
| 海量机器类通信 | mMTC | 大量设备连接 | 智慧城市、工业物联网 |

**5G关键技术**：
- 毫米波：高频段，带宽大但覆盖小
- Massive MIMO：大规模天线阵列，提升容量
- 网络切片：一张物理网切多张虚拟网，满足不同需求
- 边缘计算：数据在基站附近处理，降低延迟

**考试要点**：
- 5G频段：Sub-6GHz（覆盖好）和毫米波（速度快）
- 5G与工业互联网：5G是工业互联网的关键基础设施
- 5G与AI、物联网的关系：5G连接，物联网采集，AI处理
