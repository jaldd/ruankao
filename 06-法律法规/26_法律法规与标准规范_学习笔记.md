# 26 法律法规与标准规范

## 一、一句话理解
法律法规就是"IT行业的法律红线"——招投标、数据安全、网络安全、合同法，知道什么能干、什么不能干，考试占10-15分，必须拿下。

## 二、核心概念与框架

| 法律法规 | 核心内容 |
|---------|---------|
| 招标投标法 | 招标方式、投标要求、评标标准 |
| 政府采购法 | 采购方式、程序、质疑投诉 |
| 合同法 | 合同订立、履行、变更、解除 |
| 数据安全法 | 数据分类分级、安全保护 |
| 网络安全法 | 网络安全等级保护、关键信息基础设施 |
| 个人信息保护法 | 个人信息处理规则、权利保障 |
| 知识产权法 | 著作权、专利权、商标权 |

## 三、关键知识点逐个讲

### 知识点1：招标投标法
- **什么意思**：规范招标投标活动的法律。
- **为什么重要**：IT项目大量采用招标方式。
- **考试怎么考**：考招标方式（公开招标、邀请招标）、投标有效期、评标委员会组成。

### 知识点2：政府采购法
- **什么意思**：规范政府采购活动的法律。
- **为什么重要**：政府IT项目必须遵守。
- **考试怎么考**：考采购方式（公开招标、邀请招标、竞争性谈判、单一来源、询价）、质疑投诉程序。

### 知识点3：合同法
- **什么意思**：规范合同订立和履行的法律。
- **为什么重要**：项目合同是法律依据。
- **考试怎么考**：考合同成立要件、合同变更、违约责任、不可抗力。

### 知识点4：数据安全法
- **什么意思**：规范数据处理活动的法律。
- **为什么重要**：数据安全是国家安全的重要组成部分。
- **考试怎么考**：考数据分类分级、数据安全保护义务。

### 知识点5：网络安全法
- **什么意思**：规范网络安全活动的法律。
- **为什么重要**：网络安全是国家安全的重要组成部分。
- **考试怎么考**：考网络安全等级保护、关键信息基础设施保护。

### 知识点6：个人信息保护法
- **什么意思**：保护个人信息的法律。
- **为什么重要**：个人信息保护越来越重要。
- **考试怎么考**：考个人信息处理规则、个人权利。

### 知识点7：知识产权法
- **什么意思**：保护智力成果的法律。
- **为什么重要**：IT行业知识产权纠纷多。
- **考试怎么考**：考著作权（软件著作权）、专利权、商标权。

## 四、与其他模块的联动

### 与"采购管理"的联动
**场景**：采购管理要遵守招标投标法和政府采购法。
**处理**：采购流程要合法合规。

## 五、常见考试出题点

- **选择题一般怎么挖坑**：
  - 招标方式选择（什么情况下用邀请招标）
  - 投标有效期
  - 评标委员会组成（5人以上单数，专家不少于2/3）
  - 合同成立时间（承诺生效时）
  - 数据分类分级

- **案例分析一般怎么考**：
  - 招标过程违法分析
  - 合同纠纷分析

- **论文题一般会怎么出**：
  - 较少直接考

## 七、记忆口诀

**政府采购六方式**：
"公邀竞单询其"——公开招标、邀请招标、竞争性谈判、单一来源、询价、其他

**评标委员会**：
"五人以上单数，专家三分之二"

**合同成立**：
"要约+承诺=合同，承诺生效时成立"
