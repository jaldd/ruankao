# 06 二分查找

## 核心思路

二分查找不只是"在有序数组里找数"，更是一种 **搜索答案空间** 的思维。

**三个模板**：
1. **标准二分**：找精确目标值
2. **找左边界**：第一个 >= target 的位置
3. **找右边界**：最后一个 <= target 的位置

**万能模板**（找左边界）：
```python
left, right = 0, len(nums) - 1
while left <= right:
    mid = left + (right - left) // 2
    if nums[mid] < target:
        left = mid + 1
    else:
        right = mid - 1
return left
```

**二分答案**：当"答案具有单调性"时，二分答案+验证可行性。
