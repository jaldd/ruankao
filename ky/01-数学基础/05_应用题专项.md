## 一、工程问题

### 核心公式
```
工作总量 = 工作效率 × 工作时间
W = P × t
```

### 常见设定
- 设工作总量为 1（或最小公倍数）
- 效率 = 1/时间

### 常见题型

**1. 合作问题**
> 甲单独做需 10 天，乙单独做需 15 天，合作需几天？

```
1/(1/10 + 1/15) = 1/(3/30 + 2/30) = 30/5 = 6 天
```

**2. 先后合作**
> 甲先做 3 天，乙加入合作，又做 4 天完成。甲单独做需 20 天，乙单独做需几天？

```
甲效率 = 1/20
3 × 1/20 + 4 × (1/20 + 1/乙) = 1
3/20 + 4/20 + 4/乙 = 1
7/20 + 4/乙 = 1
4/乙 = 13/20
乙 = 80/13 ≈ 6.15 天
```

**3. 效率变化**
> 原计划 30 天完成，实际效率提高 20%，几天完成？

```
原效率 = 1/30，新效率 = 1.2/30 = 1/25
25 天完成
```

**4. 交替工作**
> 甲单独做 12 天完成，乙单独做 18 天完成。甲做 1 天，乙做 1 天，交替进行，几天完成？

```
甲效率 = 1/12，乙效率 = 1/18
两天完成 1/12 + 1/18 = 3/36 + 2/36 = 5/36
6 轮（12 天）完成 30/36 = 5/6
第 13 天甲做：5/6 + 1/12 = 11/12
第 14 天乙做：11/12 + 1/18 = 33/36 + 2/36 = 35/36
第 15 天甲做：35/36 + 1/12 = 35/36 + 3/36 = 38/36 > 1
甲只需做 (1-35/36)/(1/12) = 1/36 × 12 = 1/3 天
共 14 又 1/3 天
```

**5. 多人合作**
> 甲乙丙单独做分别需 6, 8, 12 天，三人合作需几天？

```
1/(1/6 + 1/8 + 1/12) = 1/(4/24 + 3/24 + 2/24) = 24/9 = 8/3 天
```

### 工程问题解题技巧
- **设总量为"1"**：最常用方法
- **设总量为最小公倍数**：避免分数计算（如甲 6 天乙 8 天，设总量 24）
- **效率可加**：合作效率 = 各人效率之和
- **列方程**：各人完成量之和 = 总量

## 二、行程问题

### 核心公式
```
路程 = 速度 × 时间
S = V × t
```

### 常见题型

**1. 相遇问题**
```
S = (V₁ + V₂) × t
```

**2. 追及问题**
```
ΔS = (V快 - V慢) × t
```

**3. 流水行船**
```
顺水速度 = 船速 + 水速
逆水速度 = 船速 - 水速
船速 = (顺水 + 逆水) / 2
水速 = (顺水 - 逆水) / 2
```

**例**：船顺水行 60km 用 3 小时，逆水行 60km 用 5 小时，船速和水速？
```
顺水速度 = 20 km/h
逆水速度 = 12 km/h
船速 = (20+12)/2 = 16 km/h
水速 = (20-12)/2 = 4 km/h
```

**4. 环形跑道**
```
同向追及：一圈 = (V快 - V慢) × t
反向相遇：一圈 = (V₁ + V₂) × t
```

**例**：甲乙在 400m 环形跑道上，甲速 6m/s，乙速 4m/s，同向出发，甲第一次追上乙需几秒？
```
t = 400/(6-4) = 200 秒
```

**5. 火车过桥**
```
路程 = 车长 + 桥长
```

**例**：火车长 200m，以 20m/s 通过 800m 的桥，需几秒？
```
t = (200+800)/20 = 50 秒
```

**6. 多次相遇**
```
第一次相遇：两人共走 1 个全程
第二次相遇：两人共走 3 个全程
第 n 次相遇：两人共走 (2n-1) 个全程
```

**例**：甲乙从 A、B 同时出发相向而行，第一次相遇距 A 60km，第二次相遇距 B 40km，AB 距离？
```
第一次相遇甲走 60km
第二次相遇甲走了 3×60 = 180km
甲从 A 到 B 再返回走了 AB + (AB-40) = 180
2AB = 220 → AB = 110km
```

**7. 比例法解行程问题**
> 时间相同，路程比 = 速度比
> 路程相同，时间比 = 速度的反比

## 三、浓度问题

### 核心公式
```
浓度 = 溶质 / 溶液 × 100%
溶液 = 溶质 + 溶剂
```

### 常见题型

**1. 混合问题**
> 200g 10% 盐水与 300g 20% 盐水混合，浓度？

```
(200×10% + 300×20%) / (200+300) = (20+60)/500 = 16%
```

**2. 加水稀释**
> 100g 20% 盐水加水稀释到 10%，需加水多少？

```
溶质 = 100 × 20% = 20g
20 / (100+x) = 10% → x = 100g
```

**3. 蒸发浓缩**
> 100g 10% 盐水蒸发到 20%，蒸发多少水？

```
溶质 = 10g
10 / (100-x) = 20% → x = 50g
```

**4. 十字交叉法**（快速算混合浓度）

```
浓度 a          (c-b)     份数
      \      /
        c          混合浓度
      /      \
浓度 b          (a-c)     份数
```

**例**：用 30% 和 10% 的盐水配 20% 的盐水，比例？
```
30%      10      (20-10=10)
    20%
10%      10      (30-20=10)
```
比例 1:1

**5. 反复操作型**
> 从浓度为 a% 的溶液中取出 m 克，加入 m 克水，反复 n 次后浓度？

```
最终浓度 = a% × [(V-m)/V]ⁿ
```

**例**：100g 20% 盐水，每次取出 20g 加 20g 水，3 次后浓度？
```
20% × (80/100)³ = 20% × 0.512 = 10.24%
```

**6. 两种溶液交换**
> 甲 100g 10% 盐水，乙 200g 20% 盐水，从甲取 20g 到乙，从乙取 20g 到甲，甲的浓度？

```
甲取出 20g 后：80g，溶质 = 80×10% = 8g
乙加入甲的 20g 后：220g，溶质 = 200×20% + 20×10% = 42g，浓度 = 42/220 = 21/110
乙取 20g 给甲：溶质 = 20 × 21/110 = 42/11 g
甲最终溶质 = 8 + 42/11 = 130/11 g
甲浓度 = (130/11)/100 = 13/110 ≈ 11.8%
```

## 四、利润问题

### 核心公式
```
利润 = 售价 - 成本
利润率 = 利润 / 成本 × 100%
售价 = 成本 × (1 + 利润率)
定价 = 成本 × (1 + 期望利润率)
售价 = 定价 × 折扣
```

### 常见题型

**1. 打折问题**
> 成本 100 元，定价 150 元，打 8 折出售，利润率？

```
售价 = 150 × 0.8 = 120
利润 = 120 - 100 = 20
利润率 = 20/100 = 20%
```

**2. 分段利润**
> 某商品先按 20% 利润定价，再打 9 折出售，实际利润率？

```
定价 = 成本 × 1.2
售价 = 1.2 × 0.9 = 1.08 × 成本
利润率 = 8%
```

**3. 亏盈问题**
> 甲商品盈利 20%，乙商品亏损 20%，两者售价相同，总盈亏？

```
设售价均为 120
甲成本 = 120/1.2 = 100
乙成本 = 120/0.8 = 150
总成本 = 250，总售价 = 240
亏损 10，亏损率 = 10/250 = 4%
```

**4. 多次打折**
> 某商品先提价 25%，再打 8 折，实际利润率？

```
定价 = 1.25 × 成本
售价 = 1.25 × 0.8 = 1.0 × 成本
利润率 = 0%（不赚不亏）
```

**5. 利润率与折扣关系**
> 某商品按定价打 8 折出售仍获利 20%，定价时的期望利润率？

```
设成本为 1
售价 = 1 × 1.2 = 1.2
定价 = 1.2/0.8 = 1.5
期望利润率 = (1.5-1)/1 = 50%
```

**6. 分批销售**
> 某商品成本 100 元，先按 30% 利润卖出 60%，剩余打 7 折，总利润率？

```
前 60%：售价 130，利润 30
后 40%：售价 130×0.7 = 91，亏损 9
总利润 = 30×0.6 + (-9)×0.4 = 18 - 3.6 = 14.4
总利润率 = 14.4/100 = 14.4%
```

## 五、增长率问题

### 核心公式
```
终值 = 初值 × (1 + 增长率)ⁿ
```

### 常见题型

**1. 等比增长**
> 某公司年利润 100 万，每年增长 10%，3 年后利润？

```
100 × 1.1³ = 100 × 1.331 = 133.1 万
```

**2. 恢复问题**
> 价格先涨 20%，再降 x% 恢复原价，x = ?

```
1 × 1.2 × (1-x%) = 1
1-x% = 1/1.2 = 5/6
x% = 1/6 ≈ 16.67%
```

> **注意**：涨 20% 再降 20% ≠ 原价！1.2 × 0.8 = 0.96，实际降了 4%

**3. 连续增长**
> GDP 连续两年增长，第一年增长 a%，第二年增长 b%，平均增长率？

```
(1+x)² = (1+a%)(1+b%)
x = √[(1+a%)(1+b%)] - 1
```

**4. 翻番问题**
> 某值翻一番 = 变为 2 倍
> 翻两番 = 变为 4 倍
> 翻 n 番 = 变为 2ⁿ 倍

**5. 递减问题**
> 某设备原值 100 万，每年折旧 10%，5 年后残值？

```
100 × 0.9⁵ = 100 × 0.59049 = 59.049 万
```

## 六、比例问题

### 常见题型

**1. 按比例分配**
> 甲乙丙分 120 元，比例 2:3:5，各分多少？

```
总份数 = 2+3+5 = 10
甲 = 120 × 2/10 = 24
乙 = 120 × 3/10 = 36
丙 = 120 × 5/10 = 60
```

**2. 比例变化**
> 甲乙原有存款比 3:2，甲取出 200 元后比变为 1:1，原各有多少？

```
设甲 3x，乙 2x
(3x-200)/2x = 1/1
3x-200 = 2x → x = 200
甲 = 600，乙 = 400
```

**3. 连比问题**
> 甲:乙 = 3:2，乙:丙 = 4:5，甲:乙:丙 = ?

```
统一乙：甲:乙 = 6:4，乙:丙 = 4:5
甲:乙:丙 = 6:4:5
```

**4. 比例与总量**
> 甲乙两人钱数比 5:3，甲给乙 200 元后比变为 3:5，两人原来各有多少？

```
设甲 5x，乙 3x
(5x-200)/(3x+200) = 3/5
5(5x-200) = 3(3x+200)
25x-1000 = 9x+600
16x = 1600 → x = 100
甲 = 500，乙 = 300
```

**5. 部分与整体**
> 某班男生占 60%，转走 5 名男生后男生占 55%，原有多少人？

```
设原有 n 人，男生 0.6n
(0.6n-5)/(n-5) = 0.55
0.6n-5 = 0.55n-2.75
0.05n = 2.25 → n = 45
```

## 七、年龄问题

### 核心规律
- 两人年龄差恒定不变
- 每过 n 年，每人都增加 n 岁

### 常见题型

**1. 基本年龄问题**
> 父亲今年 45 岁，儿子 15 岁，几年前父亲年龄是儿子的 5 倍？

```
设 x 年前
45-x = 5(15-x)
45-x = 75-5x
4x = 30 → x = 7.5
7.5 年前
```

**2. 年龄和问题**
> 父子年龄和为 60 岁，10 年后父亲是儿子的 2 倍，现在各几岁？

```
设父 x 岁，子 y 岁
x+y = 60
x+10 = 2(y+10)
x+10 = 2y+20
x = 2y+10
2y+10+y = 60 → 3y = 50 → y = 50/3

注意：年龄通常为整数，检查题目数据
```

**3. 三人年龄**
> 甲乙丙三人年龄和 90 岁，甲是乙的 2 倍，乙比丙大 5 岁，甲几岁？

```
设丙 x 岁，乙 x+5 岁，甲 2(x+5) 岁
x + (x+5) + 2(x+5) = 90
4x + 15 = 90 → x = 18.75
甲 = 2(18.75+5) = 47.5 岁
```

## 八、容斥原理（集合问题）

### 两集合
```
|A∪B| = |A| + |B| - |A∩B|
```

### 三集合
```
|A∪B∪C| = |A| + |B| + |C| - |A∩B| - |A∩C| - |B∩C| + |A∩B∩C|
```

### 常见题型

**1. 两集合**
> 某班 50 人，喜欢数学的 30 人，喜欢英语的 25 人，两科都喜欢的 10 人，两科都不喜欢的几人？

```
至少喜欢一科 = 30 + 25 - 10 = 45
都不喜欢 = 50 - 45 = 5 人
```

**2. 三集合**
> 某班 50 人，参加数学竞赛 25 人，物理 20 人，化学 15 人，同时参加数理 8 人，数化 6 人，理化 5 人，三科都参加 3 人，至少参加一科的几人？

```
25 + 20 + 15 - 8 - 6 - 5 + 3 = 44 人
都不参加 = 50 - 44 = 6 人
```

**3. 恰好参加两科**
> 接上题，恰好参加两科的几人？

```
恰好两科 = (8-3) + (6-3) + (5-3) = 5 + 3 + 2 = 10 人
```

## 九、最值问题

### 常见题型

**1. 均值不等式求最值**
> x > 0，求 x + 4/x 的最小值？

```
x + 4/x ≥ 2√(x × 4/x) = 2√4 = 4
当 x = 4/x，即 x = 2 时取等
最小值 = 4
```

**2. 二次函数求最值**
> 利润 = (售价-成本)×销量，售价每涨 1 元少卖 10 件，成本 50 元，原价 80 元卖 500 件，定价多少利润最大？

```
设定价 80+x 元，销量 500-10x 件
利润 = (80+x-50)(500-10x) = (30+x)(500-10x)
= 15000 - 300x + 500x - 10x²
= -10x² + 200x + 15000
x = -200/(2×(-10)) = 10
最大利润 = -10×100 + 200×10 + 15000 = 16000 元
定价 = 80+10 = 90 元
```

**3. 线性规划型**
> 生产 A 产品需 2 小时甲设备、1 小时乙设备，B 产品需 1 小时甲、2 小时乙。甲设备可用 100 小时，乙设备可用 80 小时。A 利润 300 元，B 利润 200 元，如何安排利润最大？

```
设生产 A x 件，B y 件
2x + y ≤ 100
x + 2y ≤ 80
x ≥ 0, y ≥ 0
利润 = 300x + 200y

求交点：2x+y=100, x+2y=80
4x+2y=200, x+2y=80 → 3x=120 → x=40, y=20
利润 = 300×40 + 200×20 = 16000 元
```

## 十、例题精讲

### 例 1：工程问题

> 一项工程，甲单独做 12 天完成，乙单独做 18 天完成。甲先做 3 天后，乙加入合作，还需几天完成？

**解析**：
甲效率 = 1/12，乙效率 = 1/18
甲 3 天完成 3/12 = 1/4
剩余 3/4
合作效率 = 1/12 + 1/18 = 3/36 + 2/36 = 5/36
还需 (3/4)/(5/36) = 3/4 × 36/5 = 27/5 = 5.4 天

**答案：5.4 天**

---

### 例 2：行程问题

> 甲从 A 出发以 60km/h 向 B 行驶，30 分钟后乙从 A 出发以 80km/h 追甲，乙多久追上甲？

**解析**：
甲先走 30 分钟，距离 = 60 × 0.5 = 30km
追及速度差 = 80 - 60 = 20km/h
追及时间 = 30/20 = 1.5 小时

**答案：1.5 小时**

---

### 例 3：浓度问题

> 用 10% 和 30% 的盐水配制 20% 的盐水 600g，各需多少？

**解析**：
十字交叉法：
```
10%      10      (30-20=10)
    20%
30%      10      (20-10=10)
```
比例 1:1，各 300g

**答案：10% 盐水 300g，30% 盐水 300g**

---

### 例 4：利润问题

> 某商品按定价打 8 折出售仍获利 20%，定价时的期望利润率是多少？

**解析**：
设成本为 1
售价 = 1 × 1.2 = 1.2
定价 = 1.2/0.8 = 1.5
期望利润率 = (1.5-1)/1 = 50%

**答案：50%**

---

### 例 5：增长率

> 某商品连续两次降价，每次降 10%，等价于一次降价多少？

**解析**：
1 × 0.9 × 0.9 = 0.81
等价于一次降价 19%

**答案：19%**

### 例 6：工程问题（设总量为公倍数）

> 甲乙丙单独完成工程分别需 6, 8, 12 天。甲先做 2 天，乙加入做 2 天，丙加入一起做，还需几天？

**解析**：
设总量 = 24（6,8,12 的最小公倍数）
甲效率 = 4，乙效率 = 3，丙效率 = 2

甲做 2 天：4×2 = 8
甲乙做 2 天：(4+3)×2 = 14
已完成：8+14 = 22
剩余：24-22 = 2
三人合作效率 = 4+3+2 = 9
还需：2/9 天

**答案：2/9 天**

---

### 例 7：行程问题（多次相遇）

> 甲乙从 A、B 两地同时出发相向而行，第一次相遇距 A 60km，第二次相遇距 A 多少km？AB = 150km

**解析**：
第一次相遇：甲走 60km
第二次相遇：甲走 3×60 = 180km
甲从 A 到 B（150km）再返回走了 180-150 = 30km
距 A = 150 - 30 = 120km

**答案：距 A 120km**

### 例 8：浓度问题（反复操作）

> 100g 30% 的糖水，每次倒出 10g 再加 10g 水，3 次后浓度？

**解析**：
每次操作后浓度 = 原浓度 × (90/100)
3 次后 = 30% × 0.9³ = 30% × 0.729 = 21.87%

**答案：21.87%**

### 例 9：利润问题（分批销售）

> 某商品成本 200 元，按 50% 利润定价，卖出 70% 后，剩余打 6 折，总利润率？

**解析**：
定价 = 200 × 1.5 = 300
前 70%：利润 100 × 0.7 = 70（每件利润 100）
后 30%：售价 = 300 × 0.6 = 180，亏损 20 × 0.3 = 6
总利润 = 70 - 6 = 64
总利润率 = 64/200 = 32%

**答案：32%**

### 例 10：比例问题

> 甲乙两仓库存粮比 5:3，从甲运 50 吨到乙后，比变为 3:4，甲原有几吨？

**解析**：
设甲 5x，乙 3x
(5x-50)/(3x+50) = 3/4
4(5x-50) = 3(3x+50)
20x-200 = 9x+150
11x = 350 → x = 350/11

甲 = 5 × 350/11 = 1750/11 ≈ 159.1 吨

**答案：1750/11 吨**

### 例 11：容斥原理

> 某单位 100 人，会英语 70 人，会法语 50 人，两科都会 25 人，两科都不会几人？

**解析**：
至少会一科 = 70 + 50 - 25 = 95

### 例 13：最值问题

> 某商品成本 100 元，定价 160 元，每天可卖 100 件。每降价 1 元多卖 10 件，定价多少利润最大？

**解析**：
设定价 160-x 元，销量 100+10x 件
利润 = (160-x-100)(100+10x) = (60-x)(100+10x)
= 6000 + 600x - 100x - 10x²
= -10x² + 500x + 6000
x = -500/(2×(-10)) = 25
定价 = 160-25 = 135 元
最大利润 = -10×625 + 500×25 + 6000 = -6250 + 12500 + 6000 = 12250 元

**答案：定价 135 元，最大利润 12250 元**

## 十二、易错点分析

### 易错点 1：工程问题设总量

❌ **错误**：甲 6 天乙 8 天完成，合作效率 = 1/(6+8)
✅ **正确**：合作效率 = 1/6 + 1/8 = 7/24

> **提醒**：时间是相加的倒数，效率才是直接相加

### 易错点 2：利润率分母是成本不是售价

❌ **错误**：售价 120，成本 100，利润率 = 20/120 = 16.7%
✅ **正确**：利润率 = 20/100 = 20%

> **提醒**：利润率 = 利润/成本，不是利润/售价

### 易错点 3：涨跌百分比不对称

❌ **错误**：涨 20% 再跌 20% = 不变
✅ **正确**：1.2 × 0.8 = 0.96，跌了 4%

> **提醒**：涨 a% 再跌 a%，结果一定亏

### 易错点 4：浓度问题溶质不变

❌ **错误**：100g 20% 盐水加水 50g，浓度 = 20%/1.5 = 13.3%
✅ **正确**：20/(100+50) = 13.3%（这个恰好对了）

> **提醒**：加水/蒸发时，溶质不变！只改变溶液量

### 易错点 5：行程问题追及距离

❌ **错误**：甲先走 30 分钟，追及距离 = 甲速 × 30
✅ **正确**：追及距离 = 甲速 × 0.5（30 分钟 = 0.5 小时）

> **提醒**：注意单位统一！小时 vs 分钟

### 易错点 6：比例问题设份数

❌ **错误**：甲:乙 = 3:2，甲 = 3，乙 = 2
✅ **正确**：甲 = 3x，乙 = 2x（x 是份数，需要求出）

> **提醒**：比例不等于实际值，需要列方程求 x

### 易错点 7：年龄差不随时间变化

❌ **错误**：5 年前父亲比儿子大 25 岁，现在大 30 岁
✅ **正确**：年龄差永远不变，始终大 25 岁

> **提醒**：年龄差是常数！

### 易错点 8：容斥原理忘记减交集

❌ **错误**：参加数学 30 人，参加英语 25 人，至少参加一项 = 55 人
✅ **正确**：至少参加一项 = 30 + 25 - 都参加的人数

> **提醒**：直接相加会重复计算交集

### 易错点 9：浓度十字交叉法结果

❌ **错误**：十字交叉得到的是浓度值
✅ **正确**：十字交叉得到的是**质量比/体积比**

> **提醒**：十字交叉法算出的是两种溶液的质量比

### 易错点 10：打折是乘不是减

❌ **错误**：定价 100 元打 8 折 = 100-8 = 92 元
✅ **正确**：100 × 0.8 = 80 元

> **提醒**：打 n 折 = 原价 × n/10

## 十四、本节核心结论（必背）

1. **工程**：设总量为 1，效率 = 1/时间，合作效率 = 效率之和
2. **行程**：S = Vt，相遇 S=(V₁+V₂)t，追及 ΔS=(V₁-V₂)t
3. **流水行船**：船速=(顺+逆)/2，水速=(顺-逆)/2
4. **多次相遇**：第 n 次相遇共走 (2n-1) 个全程
5. **浓度**：浓度 = 溶质/溶液，十字交叉法速算混合
6. **反复操作**：浓度 = 原浓度 × [(V-m)/V]ⁿ
7. **利润**：利润率 = 利润/成本，售价 = 成本×(1+利润率)
8. **打折**：打 n 折 = 原价 × n/10
9. **涨跌**：涨 a% 再跌 a% ≠ 原价，一定亏
10. **增长**：终值 = 初值×(1+r)ⁿ
11. **比例**：设份数 x，列方程求解
12. **年龄**：年龄差恒定不变
13. **容斥**：|A∪B| = |A|+|B|-|A∩B|
14. **最值**：二次函数顶点或均值不等式
