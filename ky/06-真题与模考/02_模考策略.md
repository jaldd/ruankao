# 模考策略

> 学习日期：2026-05-26 生成
> 用法：考前1个月开始模考，每周1-2次

---

## 一、时间分配（管理联考 180 分钟）

| 科目 | 建议用时 | 实际分配 |
|------|----------|----------|
| 数学 | 55-60分钟 | 问题求解30min + 条件判断25min |
| 逻辑 | 55-60分钟 | 形式逻辑15min + 论证逻辑20min + 综合推理20min |
| 写作 | 55-60分钟 | 论证有效性30min + 论说文30min |
| 涂卡 | 5分钟 | — |

### 做题顺序建议

**方案一（推荐）**：数学→逻辑→写作
- 优点：先做客观题，状态好
- 适合：数学基础较好的同学

**方案二**：写作→数学→逻辑
- 优点：先写完作文，心里踏实
- 适合：写作较强、怕时间不够的同学

**方案三**：数学→写作→逻辑
- 优点：数学和写作之间换脑子
- 适合：逻辑较弱的同学们

## 二、时间分配（英语二 180 分钟）

| 科目 | 建议用时 |
|------|----------|
| 阅读 Part A | 60-70分钟 |
| 阅读 Part B | 15分钟 |
| 翻译 | 20分钟 |
| 小作文 | 15分钟 |
| 大作文 | 25分钟 |
| 涂卡+检查 | 5分钟 |

### 做题顺序建议

阅读→翻译→作文
- 阅读分值最高，状态最好的时候做
- 作文有模板，放最后也来得及

---

## 三、模考执行要点

### 模考前
1. 选一个安静的上午（8:30-11:30）
2. 准备好答题卡、2B铅笔、橡皮
3. 手机静音，不查资料
4. 严格计时

### 模考中
1. 按选定顺序做题
2. 遇到难题超过3分钟先跳过
3. 每完成一个科目记录用时
4. 涂卡留5分钟

### 模考后
1. 对答案，算分
2. 分析错题原因（知识盲区/粗心/时间不够）
3. 针对性补弱
4. 调整时间分配策略

---

## 四、各科得分策略

### 数学（75分）
- 目标：45-55分
- 策略：简单题全对，中等题尽量对，难题猜
- 放弃标准：3分钟做不出就跳

### 逻辑（60分）
- 目标：40-48分
- 策略：形式逻辑全对，论证逻辑8成对，综合推理5成对
- 放弃标准：综合推理2分钟没思路就跳

### 写作（65分）
- 目标：40-45分
- 策略：论证有效性分析20+分，论说文20+分
- 关键：写完比写好重要

### 英语阅读（50分）
- 目标：30-36分
- 策略：细节题尽量全对，推断题7成对

### 英语翻译+作文（40分）
- 目标：25-30分
- 策略：翻译10+分，小作文7+分，大作文10+分
