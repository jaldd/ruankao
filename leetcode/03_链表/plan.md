# 03 链表

## 核心思路

链表题的核心技巧：

1. **虚拟头节点（dummy）**：统一头节点和非头节点的处理逻辑，必用
2. **快慢指针**：找中点、判断环、找倒数第k个
3. **画图**：链表题不画图必出错，指针指向变化要一步步画

**反转链表模板**：
```python
prev, curr = None, head
while curr:
    next_node = curr.next
    curr.next = prev
    prev, curr = curr, next_node
return prev
```
