# 逻辑公式速查

> 考前速查，建议打印贴墙

## 一、假言命题

| 命题 | 符号 | 等价 |
|------|------|------|
| 如果 A 则 B | A→B | ¬A∨B |
| 只有 A 才 B | B→A | ¬B∨A |
| 除非 A 否则 B | ¬A→B | A∨B |
| A 是 B 的必要条件 | B→A | — |
| A 是 B 的充分条件 | A→B | — |

### 推理规则

| 规则 | 说明 |
|------|------|
| A→B，A ∴ B | 肯前必肯后 |
| A→B，¬B ∴ ¬A | 否后必否前 |
| A→B，¬A ∴ ? | 否前不能否后 |
| A→B，B ∴ ? | 肯后不能肯前 |

### 等价与矛盾

| 关系 | 说明 |
|------|------|
| A→B ⟺ ¬B→¬A | 逆否等价 |
| A→B 的矛盾：A∧¬B | 前件真且后件假 |

## 二、联言与选言

| 命题 | 符号 | 含义 |
|------|------|------|
| A 且 B | A∧B | 两者同时成立 |
| A 或 B | A∨B | 至少一个成立 |
| 要么 A 要么 B | A⊕B | 恰好一个成立 |

### 德摩根律

| 公式 |
|------|
| ¬(A∧B) = ¬A∨¬B |
| ¬(A∨B) = ¬A∧¬B |

### 关键转化

| 公式 | 说明 |
|------|------|
| A∨B = ¬A→B | 相容选言→假言 |
| A→B = ¬A∨B | 假言→相容选言 |

## 三、模态命题

| 原命题 | 矛盾命题 |
|--------|----------|
| 必然 p | 可能 ¬p |
| 可能 p | 必然 ¬p |
| 必然 ¬p | 可能 p |
| 可能 ¬p | 必然 p |

口诀：**必然↔可能，肯定↔否定**

## 四、直言命题

| 类型 | 形式 | 举例 |
|------|------|------|
| 全称肯定(A) | 所有S都是P | 所有人都会死 |
| 全称否定(E) | 所有S都不是P | 所有鱼都不会飞 |
| 特称肯定(I) | 有些S是P | 有些人是左撇子 |
| 特称否定(O) | 有些S不是P | 有些人不是学生 |

### 对当关系

| 关系 | 说明 |
|------|------|
| A 与 O 矛盾 | 一真一假 |
| E 与 I 矛盾 | 一真一假 |
| A 与 E 反对 | 至少一假 |
| I 与 O 下反对 | 至少一真 |

## 五、三段论

### 有效形式

| 格 | 大前提 | 小前提 | 结论 |
|----|--------|--------|------|
| 1 | 所有M是P | 所有S是M | 所有S是P |
| 2 | 所有P是M | 所有S是M | — |
| 1 | 所有M是P | 有些S是M | 有些S是P |

### 规则

1. 中项至少周延一次
2. 前提中不周延的项结论中不得周延
3. 两个否定前提推不出结论
4. 前提有一否定，结论必否定

## 六、论证逻辑速查

| 题型 | 核心方法 |
|------|----------|
| 削弱 | 因果倒置 > 另有他因 > 切断联系 > 否定论据 |
| 加强 | 建立联系、排除他因、补充论据 |
| 假设 | 否定代入法 |
| 解释 | 兼容矛盾双方 |
| 评价 | 是/否检验法 |
| 推论 | 严格从题干推出 |

## 七、常见谬误速查

| 谬误 | 关键词 |
|------|--------|
| 以偏概全 | 少数→全部 |
| 因果倒置 | 结果当原因 |
| 相关当因果 | 同时发生≠因果 |
| 混淆概念 | 同词不同义 |
| 诉诸权威 | 专家说→一定对 |
| 滑坡谬误 | 夸大连锁 |
| 非黑即白 | 只有两种选择 |
