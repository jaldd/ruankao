# 词汇

> 开始日期：2026-05-26
> 目标：每天 50 新词 + 复习 150 词，约 30-40 分钟
> 工具：墨墨背单词 / 不背单词 App（选其一，不要换来换去）

---

## 一、背单词策略

### 核心原则

1. **认识即可，不要拼写**：阅读中只要认出词义就行，作文高频词再专门记拼写
2. **重复是记忆之母**：一个单词要见到 7-10 次才能真正记住
3. **不要纠结完美**：今天背了明天忘很正常，App 的复习算法会帮你安排
4. **利用碎片时间**：通勤、排队、睡前，每天凑够时间就行

### 优先级（按重要性排序）

```
第 1 轮（现在-7 月底）：大纲核心词 2500（高频词）
第 2 轮（8 月）：大纲边缘词 3000
第 3 轮（9-10 月）：真题超纲词 + 反复记不住的词
```

> **不要按字母顺序背**（从 abandon 开始），要按 App 的词频排序背。
