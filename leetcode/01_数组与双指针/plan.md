# 01 数组与双指针

## 核心思路

双指针是数组题的万能钥匙，核心就三种模式：

1. **对撞指针**：左右两端向中间靠拢，适用于有序数组、回文判断、两数之和
2. **快慢指针**：快指针探路，慢指针记录结果位置，适用于原地修改、去重、移除元素
3. **分离指针**：两个数组各一个指针，适用于合并、交集
