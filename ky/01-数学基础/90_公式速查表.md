## 一、算术

| 公式 | 说明 |
|------|------|
| a × b = GCD(a,b) × LCM(a,b) | 两数积=公约×公倍 |
| \|A\| + B² + √C = 0 ⟹ A=B=C=0 | 非负性 |
| a + b ≥ 2√(ab) | 均值不等式（a,b>0） |
| (a+b)/2 ≥ √(ab) | 算术≥几何 |
| a/b = c/d ⟹ (a+b)/b = (c+d)/d | 合比定理 |
| a/b = c/d = e/f = k ⟹ (a+c+e)/(b+d+f) = k | 等比定理 |

## 二、代数

### 乘法公式

| 公式 |
|------|
| (a+b)² = a² + 2ab + b² |
| (a-b)² = a² - 2ab + b² |
| a² - b² = (a+b)(a-b) |
| a³ + b³ = (a+b)(a² - ab + b²) |
| a³ - b³ = (a-b)(a² + ab + b²) |
| (a+b)³ = a³ + 3a²b + 3ab² + b³ |

| 公式 | 说明 |
