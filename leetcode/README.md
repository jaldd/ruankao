# LeetCode 90天刷题计划

> ⚡ 如果排序和查找基础不牢，先完成 **00_排序与查找（Day 0）** 再开始正式计划！

## 刷题方法论

### 核心原则：专题驱动 > 随机刷题

随机刷题效率极低，按专题刷才能形成肌肉记忆。每个专题先掌握模板，再举一反三。

### 每日流程（约1-1.5小时）

1. **先看专题核心思路**（5分钟）- 读对应文件夹的 plan.md
2. **独立做题**（30-40分钟）- 先想15分钟，没思路就看提示，不要死磕
3. **看题解对比**（15分钟）- 重点看自己没想到的角度
4. **写总结**（5分钟）- 在代码文件顶部写一句话总结核心套路

### 三刷策略

- **一刷**：按计划走，重在理解思路，不会就看答案，别硬扛
- **二刷**：隔一周重做标记的难题，重点练思路卡壳的题
- **三刷**：只做前两刷标记的高频+难题，追求15分钟内写出

## 难度分配建议

- Easy 题：快速过，5分钟内写出，练手感和速度
- Medium 题：核心刷题目标，每题控制在20-30分钟
- Hard 题：一刷选做，二刷重点攻克经典 Hard

## 题目状态标记

在代码文件顶部用注释标记：

- `✅` 一次通过
- `🔄` 看了提示才做出来，需要二刷
- `❌` 完全没思路，需要重点攻克
- `⭐` 高频面试题
