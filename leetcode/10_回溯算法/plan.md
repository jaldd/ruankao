# 10 回溯算法

## 核心思路

回溯 = DFS + 状态重置。本质是穷举所有可能，但通过剪枝减少搜索量。

**回溯模板**：
```python
def backtrack(路径, 选择列表):
    if 满足结束条件:
        结果.append(路径)
        return
    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选择
```

关键技巧：
1. **去重**：排序 + `if i > 0 and nums[i] == nums[i-1] and not used[i-1]: continue`
2. **剪枝**：提前判断不满足条件直接跳过
3. **顺序**：组合问题用 start 控制顺序，排列问题用 used 控制使用
