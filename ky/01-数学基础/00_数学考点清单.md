# 管理类联考数学考点清单

> 分值：75 分（25 题 × 3 分）
> 范围：小学、初中、高中数学，**不涉及高等数学**
> 特点：题量大（建议每题 ≤ 3 分钟），技巧性极强

---

## 一、算术（约 4-6 题，12-18 分）

| 考点 | 难度 | 重要性 |
|------|------|--------|
| 整数（奇偶、质合、整除、最大公约数/最小公倍数） | ⭐⭐ | ⭐⭐⭐ |
| 实数（绝对值、比和比例、平均值） | ⭐⭐ | ⭐⭐⭐ |
| 应用题（工程、行程、浓度、利润、增长率） | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |

**重灾区**：应用题。占比最高，变化最多，必须专项突破。

---

## 二、代数（约 6-8 题，18-24 分）

| 考点 | 难度 | 重要性 |
|------|------|--------|
| 整式与分式（因式分解、分式化简） | ⭐⭐ | ⭐⭐⭐ |
| 函数（一元二次函数、指数、对数） | ⭐⭐⭐ | ⭐⭐⭐⭐ |
| 方程（一元二次方程、韦达定理） | ⭐⭐⭐ | ⭐⭐⭐⭐ |
| 不等式（一元二次不等式、均值不等式） | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 数列（等差、等比，求和与通项） | ⭐⭐⭐ | ⭐⭐⭐⭐ |

**重灾区**：均值不等式求最值、数列综合题。

---

## 三、几何（约 5-6 题，15-18 分）

| 考点 | 难度 | 重要性 |
|------|------|--------|
| 平面几何（三角形、四边形、圆，求面积/长度） | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 解析几何（直线、圆、位置关系、对称） | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 立体几何（长方体、柱体、球体，体积/表面积） | ⭐⭐ | ⭐⭐⭐ |

**重灾区**：解析几何中圆与直线的位置关系、平面几何的阴影面积。

---

## 四、数据分析（约 5-6 题，15-18 分）

| 考点 | 难度 | 重要性 |
|------|------|--------|
| 排列组合（加法/乘法原理、捆绑、插空、分组分配） | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 概率（古典概型、独立事件、伯努利概型） | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 数据描述（平均数、方差、标准差、图表） | ⭐⭐ | ⭐⭐⭐ |

**重灾区**：排列组合（很多人高中就没学明白，这里是噩梦）。

---

## 五、学习顺序建议

```
第 1 周 ──→ 算术（先热身，应用题每天练）
第 2-3 周 ──→ 代数（方程、不等式、函数、数列）
第 4-5 周 ──→ 几何（平面 → 解析 → 立体）
第 6-7 周 ──→ 数据分析（排列组合 → 概率 → 数据描述）
第 8 周 ──→ 综合刷题，查漏补缺
```

## 六、核心公式速记（待填充）

### 代数
- 韦达定理：x₁ + x₂ = -b/a，x₁x₂ = c/a
- 均值不等式：a+b ≥ 2√ab（a,b>0，当且仅当 a=b 取等）
- 等差数列求和：Sₙ = n(a₁+aₙ)/2 = na₁ + n(n-1)d/2
- 等比数列求和：Sₙ = a₁(1-qⁿ)/(1-q) （q≠1）

### 几何
- 三角形面积：S = ½ah = ½ab·sinC
- 圆的标准方程：(x-a)² + (y-b)² = r²
- 点到直线距离：d = |Ax₀+By₀+C| / √(A²+B²)

### 数据分析
- 排列：Aₙᵐ = n!/(n-m)!
- 组合：Cₙᵐ = n!/[m!(n-m)!]
- 古典概率：P(A) = A 包含的基本事件数 / 基本事件总数
