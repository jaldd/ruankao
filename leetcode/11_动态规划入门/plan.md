# 11 动态规划入门

## 核心思路

DP的本质：**记住已求解的子问题，避免重复计算**。

解题三步：
1. **定义状态**：dp[i] 代表什么？
2. **推导转移方程**：dp[i] 如何从之前的dp值得到？
3. **确定初始值和遍历顺序**

入门DP分类：
- **一维DP**：dp[i] 只依赖 dp[i-1], dp[i-2] 等
- **背包入门**：选/不选的决策
- **打家劫舍**：相邻不能选的经典模型
