# 09 贪心算法

## 核心思路

贪心的本质：**每一步选局部最优，期望达到全局最优**。关键是证明局部最优能推导到全局最优。

常见贪心策略：
1. **排序后贪心**：按某种维度排序，然后依次选择
2. **区间问题**：按右端点排序，选最早结束的
3. **跳跃/分配**：每步尽量走最远/分配最优
