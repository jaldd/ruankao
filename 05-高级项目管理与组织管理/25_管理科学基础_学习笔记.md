# 25 管理科学基础

## 一、一句话理解
管理科学基础就是"用数学方法管项目"——运筹学、决策论、博弈论，用量化方法辅助决策。

## 二、核心概念与框架

| 核心概念 | 说明 |
|---------|------|
| 运筹学 | 用数学方法优化资源配置 |
| 线性规划 | 在一组约束条件下求最优解 |
| 决策论 | 在不确定条件下做决策 |
| 博弈论 | 研究多方决策的相互影响 |

## 三、关键知识点逐个讲

### 知识点1：线性规划
- **什么意思**：在一组线性约束条件下，求线性目标函数的最优解。
- **为什么重要**：资源分配、生产计划等场景常用。
- **考试怎么考**：考简单线性规划问题的求解。

### 知识点2：决策论
- **什么意思**：在不确定条件下选择最优方案。
- **为什么重要**：项目管理中经常面临不确定决策。
- **考试怎么考**：考决策树、期望值计算。

### 知识点3：网络计划
- **什么意思**：用网络图表示项目活动及其关系，进行优化。
- **为什么重要**：项目管理的核心工具。
- **考试怎么考**：考关键路径法、PERT。

### 知识点4：排队论
- **什么意思**：研究服务系统中排队现象的数学理论。
- **为什么重要**：IT服务管理、客服中心等场景。
- **考试怎么考**：考基本概念，复杂计算较少。

### 知识点5：存储论
- **什么意思**：研究最优库存策略。
- **为什么重要**：物资管理、供应链管理。
- **考试怎么考**：考基本概念。

## 四、与其他模块的联动

### 与"进度管理"的联动
**场景**：网络计划是进度管理的核心工具。
**处理**：关键路径法、PERT都是管理科学方法。

## 五、常见考试出题点

- **选择题一般怎么挖坑**：
  - 决策树期望值计算
  - 线性规划简单问题
  - 关键路径法计算

- **案例分析一般怎么考**：
  - 较少考
